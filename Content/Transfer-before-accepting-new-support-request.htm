Transfer before accepting a new support request

You can redirect incoming support requests to any specific technician or department. The Pending Request tab shows a list of sessions that are waiting to be accepted.

  1. Click any request to display the details in the right pane, along with the options to accept, transfer or cancel the request.
  2. Select TRANSFER REQUEST TO... to forward the request to the technician or department you specify.
  3. Add a Transfer Comment if you would like to explain the context of the request or the transfer reason.
  4. Click OK to complete the transfer.

What do you want to do?

  • Transfer a session before creating a new PIN code
  • Transfer or share a support session with another technician
  • Start an unattended support session
