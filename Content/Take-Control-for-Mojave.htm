for macOS 10.14 Mojave

With the release of macOS 10.14 (Mojave), Apple introduced a new security requirement for all remote support solutions, including . When this accessibility setting is enabled you can take control of the remote device, otherwise you can only observe when connected.

This security setting is not configurable remotely and can only be setup by an admin rights level user on the local computer.

After applying these settings, the computer is controllable using the selected remote control solutions. When these settings were applied during an active session and control problems are experienced, close the session and reconnect to the Mac.

Accessibility Access (Events) Dialog

The first time you initiate the connection to the remote Mac running macOS 10.14, the user is presented with an Accessibility Access (Events) dialog. The dialog only displays one time per app during the first connection attempt, even if the app is uninstalled then reinstalled.

To permit control from this dialog, the admin rights user must follow these steps:

  1. Navigate to System Preferences and select Security & Privacy > Privacy > Accessibility.
  2. Click the padlock in the bottom left corner to unlock and allow changes.
  3. Enter the computer's administrative account credentials.
  4. Select the check box next to the required solutions.
  5. Close the Security & Privacy window.

Security & Privacy, Privacy, Accessibility

If the option is installed on the computer but not displayed in the main Privacy window:

  1. Follow steps 1-4 above.
  2. From the Accessibility section, click the Plus icon below the main window.
  3. Use the Finder to navigate to your Applications Folder.
  4. Select the relevant Agent or Applet and click Open.

The Take Control option should now appear in the main Privacy window for selection. Close the Security and Privacy window

Enable screen recording

Within MacOS 10.15 a Screen Recording option was also added under the same menu that must be enabled as follows:

  1. Navigate to System Preferences >Security & Privacy and select Privacy from the options at the top of the window.
  2. Select Screen Recording from the options on the left.
  3. Click the padlock in the bottom left corner to unlock and allow changes.
  4. Check the box next to the Agent or Applet.
  5. Close the Security & Privacy window to save your changes.
