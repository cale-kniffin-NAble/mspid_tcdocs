Wake a remote computer

Use the Wake-on-LAN feature to bring offline computers back online.

  1. In the Console, right click in one of the computers and select Wake-on-LAN.
  2. You can wake a computer even if its marked as being online. This helps to avoid problems with the refresh of the list.
  3. The list of online Agents supporting the wake of other computers is displayed in the new window. Select the device you want to use and click OK.

    4. Select the Remember my choice box if you frequently use the same computer.

  4. Click OK in the confirmation message.
  5. When the computer is online, a small notification pops up next to the system tray.

The Agent used to broadcast the magic packet and the one requiring the wake must be on the same network if no special network configuration has been made.
