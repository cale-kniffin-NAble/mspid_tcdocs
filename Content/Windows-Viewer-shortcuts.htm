Windows Viewer shortcuts

When using the Remote Desktop tab, make sure the mouse is outside the remote desktop area for the shortcuts to work locally and not pass through the remote computer. Use the scroll lock key, and the shortcuts are executed locally, regardless of the mouse position.

Console - My Computers tab

  • Ctrl + N - New Computer/Group
  • Ctrl + D - Delete Computer/Group
  • F2 - Rename Computer/Group
  • Ctrl + M - Move Computer/Group
  • F5 - Refresh.

Viewer - Global Shortcuts

  • Ctrl + Backspace - Restart and Shutdown the remote computer.
  • Ctrl + Spacebar - Disconnect the session
  • Ctrl + Shift + F2 - Start VoIP call
  • Ctrl + Shift + F3 - Open Monitors pane
  • Ctrl + Shift + F8 - Switch to chat
  • Ctrl + Shift + F9 - Switch to Remote desktop
  • Ctrl + Shift + F10 - Switch to System Shell
  • Ctrl + Shift + F11 - Switch to File Transfer
  • Ctrl + Shift + F12 - Switch to System Information
  • Ctrl + Shift + Tab - Switch monitor

Viewer - Remote Desktop

  • Ctrl + Shift + F5 - Force the refresh of the remote image
  • Ctrl + Shift + F8 > F12 - Switch tabs
  • Ctrl + Shift + Enter - Auto hide bars in Remote Desktop
  • Ctrl + Shift + 1 - Image scale 100%
  • Ctrl + Shift + 2 - Image scale 90%
  • Ctrl + Shift + W - Image downscale to fit viewer window
  • Ctrl + Shift + G - Shared Clipboard > Disabled
  • Ctrl + Shift + J - Shared Clipboard > Bidirectional
  • Ctrl + Shift + D - Low color
  • Ctrl + Shift + M - Medium color
  • Ctrl + Shift + H - High color
  • Ctrl + Shift + T - Show remote cursor
  • Ctrl + Shift + V - View only
  • Ctrl + Shift + F - Full screen
  • Ctrl + Shift + Delete - Send Ctrl + Shift + Delete
  • Ctrl + Shift + B - Block remote keyboard and mouse
  • Ctrl + Shift + N - Blank screen
  • Ctrl + Shift + L - Lock remote operating system
  • Ctrl + Shift + K - Laser Pointer
  • Ctrl + Shift + R - Start/Stop recording video
  • Ctrl + Shift + P - Take screenshot

Viewer - File Transfer

  • Ctrl + A - Select all
  • Ctrl + O - Open
  • F5 - Refresh
  • Backspace - Up one level
  • Ctrl + N - New folder
  • F2 - Rename
  • Ctrl + M - Move to...
  • Ctrl + Shift + A - Set attributes
  • Ctrl + U - Upload selected
