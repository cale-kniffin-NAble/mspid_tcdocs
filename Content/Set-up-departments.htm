Set up Departments

Set up Departments for your technicians based on their skill set (a gold tier department made up of experienced technicians, for example). Prioritize technicians in the Department based on who is best suited to handle support requests, or send requests to all technicians in the list.

Each Department has it's own unique ID and URL to route support requests directly to them.

Watch this video to learn more about setting up Departments.

  1. Select Management > Departments > Add.
  2. Enter the name of the department and click Save.
  3. In the Department tab, the Department Name and Department ID are automatically assigned. Support requests that use the Department Link are directed to the specified department.
  4. Select Enable to activate the department.
  5. Select Technicians, and choose the employees you want to add to the department.
  6. In the Web Chat section, define the order in which assigned technicians receive web chat requests.
  7. Use the Copy Link button to copy the web chat link to the clipboard.
  8. Click Save.
