Remotely access your customers' Mac devices

After starting a remote support session with a Mac device, the Remote Viewer opens automatically on the Remote Desktop tab where you can see the Request Details from the customer's support request.

In the same window you can view the System Details containing helpful information about the device's operating system, CPU, RAM, network and disk space. You can also chat with the remote customer from the Chat window.

Open the Session Options menu to restart and shut down, or emergency reboot the remote Mac device, start a VoIP or Video call, open secure notes, or access the Computer and Session menus.

Click to access the Password Secrets Vaults during a remote support session, or select to configure and access your Passportal Vaults.

Navigate to the Connection Details icon to view the upload, download and response time of the remote Mac device.


Navigate to the Remote Desktop tab to access the following options:

  • Change visualization options during a support session
  • Send commands to the remote device
  • Interact with the remote device
