Audit Devices

allows for the auditing of one or more devices from the Admin Area. Use this feature to obtain reports about the activities performed on the selected devices, as well as detailed information about the devices themselves.

  1. From the Admin Area, select Devices.
  2. Select one or more devices from the list.
  3. Select More > Audit at the top of the page and select the audits to perform. Click Save.

    4. You can also click next to the device in the list and select Audit.

After saving the Audit, the Agent refreshes all available information for the INFO, PROCESSES, NETWORK, SERVICES, SYSTEM, APPLICATIONS, PROCESSES etc. columns at the bottom of the page.
