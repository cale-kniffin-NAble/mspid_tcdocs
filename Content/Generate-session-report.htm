Generate a Session report

The Session report provides a list of the active sessions from online technicians. Select and apply filters to sort the information contained in the report. View specific details about each support session including the problem description of the session, notes and videos, a remote system overview, and the chat transcript from the session.

  1. Navigate to Reports > Session Reports.
  2. Click Filter Title, and select filters to sort the data in the report.
  3. Select a session to view additional information in the Main and Details tabs to the right of the Session list.
  4. Select the Main tab to view details about the selected session including the Customer's ID, name, email, the problem description from the support request, and notes and videos from the support session.
  5. Select the Details tab to view information about the remote system and the chat transcription from the session.
