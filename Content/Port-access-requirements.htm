Take Control Port Access requirements

The ports identified in the tables below must be accessible for remote control connections.

If the agent has a direct TCP port configured, the same port must be open at the agent's firewall and be accessible by the viewer.

Additionally, can use the UDP transmission model to connect to devices as well as TCP. Initially, the Viewer requires access to port 1234. After the system administrator modifies the firewall to enable the identified IP addresses to communicate with the server, the ports can be random.

MacOS uses TCP Mode only.
