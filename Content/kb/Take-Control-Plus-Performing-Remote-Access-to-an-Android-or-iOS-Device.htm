: Performing Remote Access to an Android or iOS Device

Last Modified

Tue Sep 17 17:35 GMT 2019

Description

  • How to remotely connect to an Android/iOS phone or Tablet using ?

Environment

  • Android 3.2 +
  • iOS 9.0 +

Solution

  1. Login to your Windows or MacOS Take Control Technician's Console;
  2. Press START NEW SESSION;
  3. Copy the generated PIN Code and provide it to your Client;
  4. Have your client download and install the Android Applet or iOS Applet on the remote device;
  5. The client should then open the Applet and input the PIN Code previously generated and provided by the Technician;
  6. The client should then Press SEND REQUEST on the Applet;
  7. Once received, ACCEPT the incoming request to connect to the Client's device;
  8. On Android devices, a message saying "Take Control Applet will start capturing everything that's being displayed on your screen" will appear;
  9. Click the Start now button to allow screen capture;
  10. The chat section will be displayed by default, with the message "You are talking to [Technician Name]";
  11. To show live feed minimize the applet as any other android app;
  12. To terminate the session from the applet side, close the applet app as any other android app;
  • iOS system architecture does not allow screen capture and due to this it will not be possible to capture device's live feed;
    • The remote side can send screenshots of what's being displayed and access the chat feature.
