Last Modified

Wed Oct 21 18:11 GMT 2020

Description

  • Error: "The Feature was blocked at the Remote Device" when connected to a device using .

Environment

Solution

  • To resolve this, please do the following:
  1. Log in to the Admin Area (https://admin.swi-tc.com/admin_area_/home.php) using the Main or Master Account.
  2. Go to Management > Technicians.
  3. Highlight or select the technician's user account from the list.
  4. On the right side, please go to Permissions.
  5. Under Permissions, go to Console Features.
  6. Make sure that the feature that was blocked is enabled under Feature Access.
  7. Hit Save.
