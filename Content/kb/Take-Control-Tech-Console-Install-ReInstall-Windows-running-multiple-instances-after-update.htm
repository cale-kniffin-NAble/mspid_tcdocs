Take Control Tech Console Installation/Re-Installation (Windows)/Console running multiple instances after an update or install and stating I'm already logged in

Last Modified

Tue Jan 05 12:48 GMT 2021

Description

  • How to install or re-install the Take Control/ Tech Console?
  • Not receiving support requests despite the tech having permissions for it after updating;
  • Console running more services than expected and warning that I'm already logged in, only occurring after an update or re-install over an existing Console;
  • crashing after console update.

Environment

  • /;
  • Windows.

Solution

  • If the issue started happening after a Take Control console update, please proceed as follows:
    1. Uninstall the existing Take Control Technician Console;
    2. If still existing, delete the folder: C:\Users\USERNAME\AppData\Local\BeAnywhere Support Express\Console
    3. Reboot the computer (OPTIONAL);
    4. Install the latest Take Control Tech Console version;
      • You can get the latest version from Take Control: Downloads.
      • Or you can get it from the Admin Area:
        1. Login to the Admin Area.
        2. Navigate to the Downloads section and download the Technician Console installer.
