: How to set agent customer details

Last Modified

Thu Feb 01 16:12 GMT 2018

Description

  • How to set customer details on an Agent?

Environment

Solution

  1. Login on your Tech Console
  2. Select the MY COMPUTERS tab
  3. Choose the desired agent on the list
  4. Set the details on the form at hand (right side)
  5. Click SAVE
