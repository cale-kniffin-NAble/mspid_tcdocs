Description

  • What are the Take Control Standalone remote support session options to access client devices?

Environment

    • Applet
    • Agent

Solution

  • There are two types of remote support sessions available through Take Control Standalone: the Applet and the Agent
  • Applet (Attended Support) sessions allows remote support with no additional application installed on the end user?Ts machine.
    • More information on Applet sessions can be found in the User's Guide here.
  • Agent sessions require an application to be installed on the end user?Ts machine but allows for unattended sessions as well as more advanced settings to be applied.
    • More information on Agent sessions can be found in the User's Guide here.
