: How to enable/disable File Transfer access for End Users

Last Modified

Thu Apr 09 16:08 GMT 2020

Description

  • This article addresses the possibility of enabling or disabling File Transfer for end users on

Environment

Solution

  1. Login to the Admin Area;
  2. Go to Management > Technicians;
  3. Select the intended End User;
  4. Tick/Untick the checkbox Access FTP;
  • This must be done by a Technician with sufficient privileges to perform these changes.
