Take Control displaying Remote Desktop in grayscale

Last Modified

Wed Nov 14 15:02 GMT 2018

Description

  • The Remote Desktop is in grayscale. How can I change the screen to feature color?

Environment

    • .
  • :
    • Take Control.
  • /.

Solution

  • The View options for the Viewer's Remote Desktop tab may be set to Grayscale to lower resources used, please review and change them as follows:
    1. Select the Remote Desktop tab;
    2. Select the View menu;
    3. Select the Color sub-menu;
    4. Set the color option to either Low Color, Medium Color, High Color or True Color.
      • Note that each of these options is increasingly more visually demanding than the last and may impact performance.
  • If the Remote Desktop display is displayed in gray-scale even with a low/medium/high/true color view option and also displaying in gray-scale to the end user on a Windows 10 device, please attempt to toggle between Grayscale and Full Color display by using the CTRL+Windows Key+C key combination or via the Color Filter option
