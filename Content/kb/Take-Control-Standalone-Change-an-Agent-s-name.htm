: Change an Agent's name

Last Modified

Thu Aug 01 08:53 GMT 2019

Description

  • How to change an Agent's name.
  • How to change a Device's name.

Environment

  • \

Solution

  • via Take Control Tech Console:
  1. Login to your Take Control Technician Console application;
  2. Select the MY COMPUTERS tab;
  3. Right-Click the desired Agent from the My Computers list;
  4. Select Rename;
  5. Type the new device name;
  6. Press Enter.
  • via Admin Area:
  1. Login to the Admin Area;
  2. Navigate to Devices;
  3. Select the intended device from the devices list;
    • A chart will open below with the GENERAL tab opened by default;
  4. Under that tab, change the field Computer Name;
  5. Click Save.
