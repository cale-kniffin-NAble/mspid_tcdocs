: Support for file transfer from mapped drives

Last Modified

Fri Jul 19 10:47 GMT 2019

Description

  • Does File Transfer support Mapped Drives?
  • Why can't I see mapped drives during Agent sessions?

Environment

  • /.

Solution

  • You are able to view Mapped Drives to a given user during Applet (attended support sessions) in the File Transfer tab;
  • It is not possible to view Mapped Drives during Agent (unattended support sessions) as you won't be able to see any network drives as they aren't actually mapped for the system account that is the one that the agent/service uses.
  • Although not appearing by default, you can manually map the network drives for the system account (which would allow them to be visible during Agent sessions).
  • For more information on mapped drives and how to manually map them to the system account, please consult the following link: https://superuser.com/questions/753400/map-network-drive-for-the-system-user-in-windows
