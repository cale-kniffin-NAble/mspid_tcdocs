Create new remote session from Technician's console

Last Modified

Wed Nov 30 21:44 GMT 2016

Description

This article details the steps on how to create session on the Technician's console to initiate a remote session.

Environment

All versions

Solution

  1. Log in to the Technician Console.
  2. Click the Start New Session Button.
  3. If that option does not display on the console, ensure your login has sufficient privileges by verifying it from the Admin Console (Technicians > Permission > Console Features).
  4. After successfully creating a new session, choose one of the two options to initiate remote session.
