Cannot set Chrome to remember as the default application

Last Modified

Tue Mar 10 10:18 GMT 2020

Description

  • When launching Take Control using Chrome, there's no option to set it as default.
  • Will be prompted to "Open " each time a Take control session is launched:

Environment

  • Take Control
  • Chrome v77

Solution

  • Chrome no longer displays the "Always open these types of links in the associated app" option when initiating a Take Control session
  • This is not an issue with //Take Control
  • This can be set using IE or Firefox as this is only an issue with Chrome version 77 and above.
  • Further documentation can be found on this here:
    • https://bugs.chromium.org/p/chromium/issues/detail?id=982341#c2
