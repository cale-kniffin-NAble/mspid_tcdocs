Take Control: Admin Area/Deferred Support Request interface/Survey not loading correctly on Internet Explorer

Last Modified

Fri Mar 22 16:57 GMT 2019

Description

  • Unable to login through the Admin Area on Internet Explorer 11 or previous versions!
  • Admin Area components not loading correctly on Internet Explorer!
  • Windows for Surveys or Deferred Support Requests are not displaying correctly for the End User when using Internet Explorer!

Environment

  • /.

Solution

  • There are known compatibility issues with Internet Explorer as it is no longer a supported browser. As the Admin Area and it's features make use of HTML 5.0, please consider using an alternative supported browser, such as Chrome, Firefox, Edge or Opera.
