Take Control: How to change request details and have them e-mailed in a Session Report

Last Modified

Wed Jun 19 13:34 GMT 2019

Description

  • How can I change Request Details (such as Notes and Problem Descriptions) mid-session?
  • Can I then e-mail an updated session report to client's or another e-mail address?

Environment

  • /:
    • Take Control Console 7.00.00 or greater.

Solution

  1. Perform a remote session to the intended device;
  2. Click the System Details/Request Details icon (2 squares and a rectangle), located on the top right of your Viewer;
  3. Select the REQUEST DETAILS tab;
  4. Edit the text fields as intended;
  5. Enable Send report to client when session ends;
  6. Fill out the Email report to field as intended;
    • Note that, by default, this will be automatically filled with the information from the Email field (the client's Email).
  7. SAVE when done (near the bottom right of the Request Details prompt);
  8. When the session ends, an e-mail containing a Session Report link will be automatically sent the specified address - that link will reveal a full session report, including the information in the text fields mentioned in step 4.
