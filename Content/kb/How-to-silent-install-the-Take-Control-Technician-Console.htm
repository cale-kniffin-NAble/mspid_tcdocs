How to silent install the Take Control Technician Console

Last Modified

Tue 12 Jan 17:45 GMT 2021

Description

  • Instructions on how to silent install the Take Control Technician Console;

Environment

  • /

Solution

  1. Open the Windows command prompt and run the Take Control Console installer with the /S parameter, as follows:
    • TakeControlTechConsole-Stable.exe /S
  • Both from the Web and Desktop option;
  • To change the default installation path to a custom one, see [%=General.ProductName_tc-sa%]: How to set a custom path for the Technician Console.
