: How to enable Monitoring for a device

Last Modified

Thu Jan 10 17:38 GMT 2019

Description

  • How to enable Monitoring on a remote Agent.

Environment

Solution

  1. Login to your Admin Area;
  2. Navigate to Devices;
  3. Select the desired device/agent from the list;
  4. On the chart below, select the GENERAL tab;
  5. Slide the Enable monitoring option to ON;
  6. Press Save (bottom right corner).
