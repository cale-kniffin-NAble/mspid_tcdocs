Take Control (All Versions): How to enable/disable Keyboard Mapping?

Last Modified

Wed Dec 16 17:42 GMT 2020

Description

  • How to enable/disable keyboard mapping through Take Control?
  • How to ensure local characters are typed correctly on the remote end of a Take Control session?
  • My keyboard is not typing certain alphanumeric characters on the remote end or typing them incorrectly.

Environment

  • /;
  • :
    • Take Control.
    • .
  • Windows;
  • macOS.

Solution

  • During the remote session (on the Viewer's end):
    1. Left-click on Remote Desktop;
    2. Hover over the Interactions menu;
    3. Enable or Disable the Enable Keyboard Mapping option, as intended, by left-clicking on it.
      • While this option is enabled, local keyboard inputs will be attempted to be matched directly on the remote end - this should be used when the Viewer is assuming the remote keyboard layout as different from the local keyboard.
