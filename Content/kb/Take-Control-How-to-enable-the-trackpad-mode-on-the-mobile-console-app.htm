Take Control: How to enable the trackpad mode on the mobile console app

Last Modified

Sun Mar 17 21:03 GMT 2019

Description

  • How to enable trackpad mode on the Take Control mobile app console.
  • How to enable the remote cursor.

Environment

Solution

  1. Login on the Take Control console app;
  2. Launch a remote session to the intended device;
  3. Select the Remote Desktop icon;
  4. On the top left corner click the Menu icon (looks like 3 horizontal lines);
  5. Slide to the right the option Trackpad Mode;
