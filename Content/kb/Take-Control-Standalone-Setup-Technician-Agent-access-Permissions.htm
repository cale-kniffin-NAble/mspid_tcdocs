: Setup Technician Agent access Permissions

Last Modified

Tue May 21 08:34 GMT 2019

Description

  • How to limit which Agents a Technician has access to?
  • How to setup a Technician's permissions to create/delete/move Agents?
  • How to setup a Technician's permissions to create/delete/move Agent Groups?

Environment

  • /.

Solution

  1. Login to your Admin Area
    • Please ensure you log in with sufficient privileges (using the Main Admin login is preferred).
  2. Navigate to Management > Technicians;
  3. Select the intended Technician's Profile from the list;
  4. Select the Permissions tab;
  5. Select the GLOBAL tab;
  6. Set the options as intended (under My Computers Management):
    • Access to My Computers: Enable/Disable access to the My Computers tab (and all access to Agents);
    • Can Edit Computer Details: Enable/Disable the Technician from being able to edit Agent Customemr details (Name, Email, Customer Number, Description, etc.);
    • Add: Enable/Disable the Technician from creating new Agent Installers/New Computer Groups;
    • Delete: Enable/Disable the Technician from uninstalling Agents/deleting Computer Groups;
    • Move: Enable/Disable the Technician from moving Agents/Groups;
    • Edit: Enable/Disable the Technician from renaming Agents/Groups;
    • Select Groups: Select which Groups of Agents the Technician will be able to access.
      • The Technician will ALWAYS be able to access Agents without Groups (in the ./root group) as long as they have Access to My Computers enabled.
  7. Save when done;
  8. The Technician will need to login to the Console after these changes are made to pickup the new settings.
