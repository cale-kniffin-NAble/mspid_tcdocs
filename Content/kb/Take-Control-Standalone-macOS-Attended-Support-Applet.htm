: macOS Attended Support (Applet)

Last Modified

Thu Oct 29 18:13 GMT 2020

Description

  • How to obtain and execute the macOS Applet

Environment

  • /
  • macOS

Solution

  1. Generating a new session PIN Code (As a Technician):
    1. Open your Take Control Tech Console application
    2. Press Start New Session
    3. Provide the 9 digit PIN Code to your client
  2. Obtaining and running the macOS Applet (As a Client):
    1. Navigate to the Start Control webpage
    2. Select the Mac Version tab
    3. Press the Download Applet button
    4. Execute the Applet
    5. Input the previously generated 9 digit PIN Code to perform a remote support request;
    • If the first 3 digits are zeros don't bypass them, they need to be included, otherwise the PIN code will be invalid.
