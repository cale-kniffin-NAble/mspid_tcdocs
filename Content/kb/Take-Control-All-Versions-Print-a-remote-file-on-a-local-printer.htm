Take Control (All Versions): Print a remote file on a local printer

Last Modified

Mon Nov 25 17:18 GMT 2019

Description

  • While on a Take Control session, I want to print a file from the remote device on a local printer

Environment

  • Take Control

Solution

  1. Start a session to the desired remote device (where the file is located);
  2. In the Viewer window, left click the FileTransfer tab;
  3. Locate the file on the Remote File List;
  4. Right click on the remote file and select Print;
    • Alternative, you may use the hotkey: CTRL+P.
  5. Approve the file download by pressing Yes;
  6. The print interface (according to your local "Open with" settings) should now appear.
