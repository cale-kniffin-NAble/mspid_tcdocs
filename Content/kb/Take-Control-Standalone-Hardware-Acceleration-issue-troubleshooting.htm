Hardware Acceleration issue troubleshooting

Last Modified

Wed Feb 19 13:14 GMT 2020

Description

  • Slow Remote session to Take Control Agent with good ping/low ms values;
  • Slow typing;
  • Slow clicking;

Environment

  • /
  • Windows OS
  • Known issued graphics adapters:
    • Matrox G200eh;
    • Matrox G200eW;
    • Matrox G200eR;
    • Matrox G200e (Server Edition);
    • Matrox G200e (ServerEngines);
    • ATi ES 1000;
    • ATi Rage XL.

Solution

  • For the following adapters: Matrox G200eh, Matrox G200eW, ATi ES 1000 ATi, RAGE XL:
    1. Open the Take Control Agent Configuration program on the remote device with Admin Privileges;
    2. Select the SETTINGS menu;
    3. Check the Optimize desktop settings when in session;
    4. Check the Disable Hardware Acceleration setting;
    5. APPLY the changes;
    6. Try a new connection;
    7. Should this not resolve the issue, a direct change on the remote device's Windows settings will be required:
      1. Open the Control Panel;
      2. Go to Appearance > Personalization > Display;
      3. Click Adjust screen resolution and then Advanced Settings;
      4. Browse to the Troubleshoot tab and click Change Settings (you will get the UAC prompt here, accept it);
      5. Move the slider from Full to None;
      6. Apply the changes and try a new connection;
  • For the following adapters: Matrox G200eR, Matrox G200e (Server Edition), Matrox G200e (Server Engines):
    1. Open the Server Configuration program on the remote device with Admin Privileges;
    2. Select the SETTINGS menu;
    3. Uncheck the Disable Hardware Acceleration setting (we'll WANT to have Hardware Acceleration enabled);
    4. Uncheck the Optimize desktop settings when in session;
    5. Check the Limit Image Capture;
    6. APPLY the changes;
    7. Open the Control Panel;
    8. Go to Appearance > Personalization > Display;
    9. Click Adjust screen resolution and then Advanced Settings;
    10. Browse to the Troubleshoot tab and click Change Settings (you will get the UAC prompt here, accept it);
    11. Move the slider from None to Full;
    12. Apply the changes and try a new connection.
