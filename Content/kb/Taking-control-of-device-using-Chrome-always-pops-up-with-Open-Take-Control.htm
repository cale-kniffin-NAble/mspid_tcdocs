Taking control of a device using Chrome always pops up with "Open ?"

Last Modified

Thu Oct 31 13:54 GMT 2019

Description

  • Taking Control of a device using the latest Chrome version will always show the following:
  • The checkbox which was in previous Chrome versions - "Always open these types of links in the associated app" - is gone

Environment

  • Take Control

Solution

  • Chrome has removed this tickbox in the latest version of its browser
  • This was removed due to a vulnerability
  • More information can be found in this Chromium article:
    • https://bugs.chromium.org/p/chromium/issues/detail?id=982341#c2
  • Since this is a change in the browser behavior there is no workaround available
