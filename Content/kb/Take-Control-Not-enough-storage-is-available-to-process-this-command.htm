Take Control: Not enough storage is available to process this command

Last Modified

Wed 13 Jan 13:02 GMT 2021

Description

  • Agent not allowing connections to remote device despite being ready/available;
  • Error: The server may be down or unavailable please try again later;
  • Remote Agent logs reveal following:
    • [TXMLParser::TXMLParser] - Exception creating XML parser - Not enough storage is available to process this command

Environment

  • /;
  • :
    • Take Control.

Solution

  • The error in question indicates a lack of Kernel resources for the parser to proceed with the required command;
  • Note that this issue is not specific to Take Control but is a Windows error indicating that there are insufficient resources available to perform the requested operation (often linked to an IRPStackSize limit).
  • Rebooting the device will resolve the issue as it will liberate the resources and prevent resource exhaustion.
  • Please consider the following Microsoft articles on the issue:
    • https://support.microsoft.com/en-us/help/106167/error-message-not-enough-server-storage-is-available-to-process-this-c
    • https://support.microsoft.com/en-us/help/3101545/not-enough-server-storage-is-available-to-process-this-command-error-w
    • https://support.microsoft.com/en-us/help/2877056/not-enough-storage-is-available-to-process-this-command-error-when-a-n
