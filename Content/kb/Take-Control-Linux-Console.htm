Take Control: Linux Console

Last Modified

Thu Mar 19 13:18 GMT 2020

Description

  • Can the console be installed in Linux and/or can I control other computers from the web console from a Linux Distro?

Environment

  • /.

Solution

  • The Take Control Console does not currently support Linux based operating systems.
  • You can make use of the Console in Windows, macOS, Android and iOS, please check your Downloads section of the Admin Area for all available Console versions.
