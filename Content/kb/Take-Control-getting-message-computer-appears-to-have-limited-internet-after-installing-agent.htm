Take Control: getting the message "This computer appears to have limited internet connectivity" after installing the agent

Last Modified

Mon Jan 28 14:24 GMT 2019

Description

  • After installing the Take Control Agent the following message is shown: "This computer appears to have limited internet connectivity".

Environment

  • /

Solution

  • This message appears when the Take Control Agent communication requirements are not being met, meaning our domains are not being allowed, either by a Firewall (networkwise or endpoint level), Content Filter, AV, endpoint Security Software, etc.
  • For more information, see [%=General.ProductName_tc-sa%]: Ports and Domains/Firewall and AV Exclusions.
