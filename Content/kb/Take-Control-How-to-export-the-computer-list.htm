Take Control: How to export the computer list

Last Modified

Wed Feb 13 13:56 GMT 2019

Description

  • How to create a report with all the agents currently installed?
  • How to create a report with all the agent versions in use?

Environment

  • /.

Solution

  1. Login to your Admin Area;
  2. Select the Devices menu from the left-hand pane;
  3. Click the Export List button near the top right corner;
  4. A .csv file will automatically download to your device containing the device list and their Agent details (LastSeen, Version, etc.).
