Take Control (All Versions): How to change the size of the remote desktop

Last Modified

Thu Jan 30 10:01 GMT 2020

Description

  • Remote Desktop resolution is too small or large
  • How to Zoom in or out of the Remote Desktop?
  • How to change the Remote Desktop resolution during a session?

Environment

  • /
  • Take Control

Solution

During a remote session:

  1. Select the Remote Desktop tab
  2. Select the View menu
  3. Use one of the following options as needed:
    • Size: Set the desired scale (zoom) of the remote desktop display (100% being 1:1)
    • Change Remote Resolution: Alter the remote desktop resolution to a desired size (usually the same as your local resolution)
    • Full Screen: Take the Remote Desktop into Fullscreen (you can enter/leave Full Screen by using the hotkeys Shift+Ctrl+F)
