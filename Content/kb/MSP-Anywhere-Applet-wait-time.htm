: Applet wait time

Last Modified

Tue Oct 31 19:17 GMT 2017

Description

  • How to extend or reduce the default timeout for Applet session requests?
  • How to allow clients to wait more or less time when running the attended support request applet and waiting for a technician?

Environment

Solution

  1. Login to your Admin Area
  2. Navigate to Profile > Applet Setup
  3. Select the 6.Messages 3 out of 4 tab
  4. Set the Timeout value as desired (in seconds)
  5. Save when done
  • Note that this does not prevent your client from, for example, closing the Applet and cancelling the request
