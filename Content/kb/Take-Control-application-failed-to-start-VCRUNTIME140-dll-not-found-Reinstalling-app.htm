Take Control: This application has failed to start because VCRUNTIME140.dll was not found. Re-installing the application may fix the problem.

Last Modified

Mon Nov 12 10:35 GMT 2018

Description

  • After an install/update, the Agent can't start and shows the following:
    • Error: This application has failed to start because VCRUNTIME140.dll was not found. Re-installing the application may fix the problem.

Environment

Solution

  1. Login to your Take Control Tech Console;
  2. Select the MY COMPUTERS tab;
  3. Press the ADD COMPUTER... button;
  4. Fill out the device name;
  5. Press the GENERATE LINK button;
  6. Access and download the Agent installer from the generated link on the issued device;
  7. Run the installer over the existing Agent installation.
