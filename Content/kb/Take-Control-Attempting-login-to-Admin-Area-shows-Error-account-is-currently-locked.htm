: Attempting to login to the Admin Area shows Error: "This account is currently locked out. Please contact your administrator."

Last Modified

Wed 13 Jan 13:05 GMT 2021

Description

  • After attempting to login several times, I'm now getting the following Error: "This account is currently locked out. Please contact your administrator."

Environment

  • /

Solution

  • This error is related to our login feature, in particular, that there were too many failed attempts to enter the correct username and password - locking the account's login for 30 minutes.
  • This is a security requirement for the login, you should be able to re-attempt to login within 30 minutes.
  • Ensure your password is correctly set. For more information, see [%=General.ProductName_tc-sa%]: How to reset Password.
  • If, after the 30 minutes have passed or you've performed the password recovery procedure, and you're able to login to the Admin Area but not the Console, review you console version. For more information, see [%=General.ProductName_tc-sa%]: Console install/update/Technician getting Login/Password failed error after associating with SSO.
