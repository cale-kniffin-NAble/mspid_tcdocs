: Add or Change a Technician's photo

Last Modified

Tue Nov 12 13:04 GMT 2019

Description

  • How to change a Technician's photo from the Admin Area?
  • How to upload an avatar for a Technician to display during Remote Session chats?

Environment

  • /.

Solution

  1. Login to your Admin Area;
    • You may also access this Admin Area from your Console by pressing "Admin Area" to the top of the console window
    • You will need permissions to access the Admin Area, should you not have the required permissions - request your account's main administrator to provide them or to perform the change themselves
  2. Navigate to Management > Technicians;
  3. Select the intended Technician's Profile;
  4. Select the Change Picture option;
  5. Select the intended image and press Open;
    • The file's format must be JPG/JPEG and not exceed 200KB in size.
  6. Save the changed Technician's Profile (lower right corner).
