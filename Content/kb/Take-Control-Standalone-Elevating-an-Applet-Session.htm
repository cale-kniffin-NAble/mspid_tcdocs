: Elevating an Applet Session

Last Modified

Tue Nov 12 15:07 GMT 2019

Description

  • How to elevate the permissions of an Applet Session?

Environment

  • /.

Solution

  1. Select the Session Options menu (top-right, appears like a downward arrow within a circle) in the Viewer window
  2. Select Computer
  3. Select Request Administrator Privileges;
  4. You will be prompted to select either:
    • The client will provide Admin Credentials, if needed.
    • I will provide Admin Credentials: Input both the Remote Admin credentials (username and password).
  5. Press OK;
  6. The End User will need to approve an elevation prompt and a UAC prompt - restarting the Applet with Elevated Privileges.
