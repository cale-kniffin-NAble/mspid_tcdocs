: Default storage location for Take Control session recordings

Last Modified

Tue Apr 28 16:11 GMT 2020

Description

  • Where to find the raw recording FLV files prior to their upload?
  • Where is the Session Recording local cache on the Technician's computer?

Environment

  • Windows
  • macOS

Solution

  • While pending their automatic upload, recordings will be stored in a local cache, meaning in the workstation from where the Technician performed their sessions ;
  • Session recordings pending upload can be found under the following path:
    • Windows: C:\Users\Bruce\AppData\Local\BeAnywhere Support Express\Console\FLV (C:\Users\Username\AppData\Local\BeAnywhere Support Express\Console\FLV)
    • macOS: /Users/<user>/Documents/MSP Anywhere Files/Console/FLV
  • Please note that any tampering with these files may compromise their upload;
  • The videos will be in an FLV format, which can be played by Media Players with the relevant Codec (VLC media player for instance)

Article Number: 122002

Internal notes: !! Please do not add RMM or N-central Take Control to this list. This article is relevant EXCLUSIVELY to standalone Take Control/' default recording cache. !!
