Description

  • Can I setup a way to download Take Control Applets using a call to a webservice?
  • Is there an API I can use to start New Take Control Applet Downloads for Attended Sessions?
  • Where can I find my UID and manual to use the New Session API for Take Control?

Environment

  • Plus (Standalone)

Solution

  • The New Session API enables the generation of Attended Support Applets using a call to a webservice/API. The webservice is capable of receiving optional parameters that are used to pre-configure the support request, like the name and the e-mail of the customer, the client number or the problem description. These values will then be sent to the technician?Ts console, allowing the easy identification of the support request.
  • Additional information, requirements and manual for use of this API can be found as follows:
    1. Login to your Admin Area;
    2. Navigate to Profile > APIs;
    3. Select tab START APIs (selected by default);
    4. There you'll find your UID and by clicking the Help button you'll be directed to the relevant documentation with detailed instructions on how to use this functionality.
