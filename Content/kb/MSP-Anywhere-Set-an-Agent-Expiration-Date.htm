: Set an Agent Expiration Date/Automatically remove Agent

Last Modified

Wed May 09 13:30 GMT 2018

Description

  • How to set an Agent install to automatically remove itself from the client's device and the My Computers list after a certain time?
  • How to set an Agent Expiration Date?

Environment

Solution

  1. Login to your Console Application;
  2. Select the My Computers tab;
  3. Select the ADD COMPUTER button;
  4. Fill out the form on the right as intended (Computer Name and Device Type are mandatory);
  5. De-select Computer never expires and set the Computer Expiration date;
  6. Press GENERATE LINK;
  7. Access (or have your client access) the generated link to download the installer (if the client is on a macOS/OSX device, they'll also need the generated PIN Code);
  8. Run (or have the client run) the downloaded installer;
  9. Once the expiration date is reached, the device will be automatically removed from your My Computers list and the Agent will be uninstalled from the client's device.
