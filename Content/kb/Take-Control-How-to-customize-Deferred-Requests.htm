Take Control: How to customize Deferred Requests

Last Modified

Tue Mar 12 18:10 GMT 2019

Description

  • How to customize Deferred Requests.
  • How to define an introductory text.
  • How to add questions to your deferred support request form.

Environment

Solution

  • To define an introductory text:
  1. Login to your Admin Area;
  2. Navigate to Management > Deferred Requests;
  3. In the text box below the "Intro" section, write the intended introductory text;
  4. Click Save.
  • To add questions to your deferred support request form:
  1. Log in to the Admin Area;
  2. Navigate to Management > Deferred Requests;
  3. In the "User Content" section, click the Add button;
    • Define the question, the answer type, activate/deactivate the option "Check before submit" or even the question itself and set the position in which the question should be shown (remember that you can have up to 10 questions);
  4. Press the Add button;
