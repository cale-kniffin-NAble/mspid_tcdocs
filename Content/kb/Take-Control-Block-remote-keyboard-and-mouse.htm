Take Control: Block remote keyboard and mouse

Last Modified

Wed Sep 30 12:42 GMT 2020

Description

  • How to keep the remote input from interfering during a session?
  • Can I block the remote keyboard/mouse input? How?

Environment

  • /.

Solution

  • You can disable remote input during a session as follows:
    1. Select the Remote Desktop tab;
    2. Select the drop-down menu to the right of Remote Desktop (looks like a downward facing arrow);
    3. Select the Commands sub-menu;
    4. Enable Block Remote Keyboard and Mouse (or use the hotkey combination Shift+Ctrl+B).
  • Note: This feature is not available to End Users.
