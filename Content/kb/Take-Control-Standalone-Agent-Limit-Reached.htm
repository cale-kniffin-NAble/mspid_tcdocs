: Agent Limit Reached

Last Modified

Fri Jan 11 12:27 GMT 2019

Description

  • Unable to install the Agent due to Agent Limit error
    • Error: Agent limit reached

Environment

Solution

  • (Windows only) Please ensure that the remote device has a stable Microsoft XML Parser instalation
    • Download and execute the following Microsoft package on the remote device: Microsoft XML Core Services
  • Generate a fresh installer - as the previous one may have reached it's installation limit
    1. Login to your Take Control Tech Console application
    2. Select the My Computers tab
    3. Press the Add Computer button (bottom left corner)
    4. Fill out the form to the right
      • Pay special attention to the Number of Installations field - which can be set to any value between 1 and 100 installations
    5. Press GenerateLink
    6. Download the Agent Installer from the Generated Link
    7. Attempt the installation with the newly generated Agent Installer
  • Verify the total number of Agents you may install
    1. Login to your Admin Area
    2. Click on the Tech's name on the top right corner
    3. Click on Licensing Information
    4. On the Overview page verify the total number of available Licenses and Agents
      • Note that each license entitles you to 500 Agents (2 licenses to 1000 Agents, 3 licenses to 1500 Agents, etc.)
      • Note that each Take Control license entitles you to 50 Agents (2 licenses entitles you to 100 Agents, 3 licenses entitles you to 150,etc.)
    5. Uninstall any excess Agents in order to liberate room for future installations

Article Number: 128011

Internal notes: To the top of the page verify the total number of Installed Agents under the Computers field (includes both Online and Offline Agents)The combined number of Online and Offline Agents is the total number of Agent installations This number will need to be lower than your allowed total per license in order to perform any new installations
