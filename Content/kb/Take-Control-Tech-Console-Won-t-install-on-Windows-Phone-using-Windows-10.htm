Take Control Tech Console Won't install on Windows Phone using Windows 10

Last Modified

Tue Nov 20 13:32 GMT 2018

Description

  • Unable to install the Take Control Tech Console on my Windows Phone device - even though it uses Windows 10.

Environment

  • /.

Solution

  • At this time, we do not have a console made specifically for Windows Phone devices (even those running Windows 10).
  • This is due to the hardware used on Windows Phone devices (in particular the ARM architecture used by the processor) not supporting our current tech console.
  • Note that this does not apply to Microsoft Surface devices - which can use the standard Windows console.
