: How to generate an Agent Installer (Unattended Support)

Last Modified

Tue Feb 11 11:39 GMT 2020

Description

  • How to generate the installer for Take Control's Agent for an unattended support session?

Environment

  • /
  • Windows
  • MacOS
  • Linux

Solution

  • You can generate an installer or obtain a default installer as follows:
    • From the Console application:
      1. Login to your Console application;
      2. Select the MY COMPUTERS tab;
      3. Press ADD COMPUTER;
      4. Fill the form as desired;
        • Make sure you fill the Computer Name and Device Type fields, the only ones mandatory;
      5. Press GENERATE LINK;
      6. Download the Installer from the generated link and run the installer on the target device.
    • From the Admin Area (Default Installer):
  1. Login to your Admin Area;
  2. Select Downloads;
  3. Download the agent installer by clicking the desired OS button.
  • Please note that this installer will have default settings (an install limit of 100 devices and will not make use of a custom device name)!
  • The installer
  • From the Admin Area using Remote Control Mode:
  1. Login to your Admin Area;
  2. Go to Devices;
  3. Click the + Add device button;
  4. Fill the form as desired;
  5. Click the Save button;
    • For this you need to have the Remote control mode turned on,otherwise a message will show requesting to enable it by clicking the Remote Control Mode icon, on the top right corner of the Admin area page
  6. Use the PIN code option or the URL option, what best suits your needs.
  • You can find additional information on creating unattended install links and adding devices in the User Guide: [%=URLs.TakeControlHelpURL%]Unattended-support.htm
