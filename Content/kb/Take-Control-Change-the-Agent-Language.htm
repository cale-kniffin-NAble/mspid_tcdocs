Take Control: Change the Agent Language

Last Modified

Mon Feb 18 17:56 GMT 2019

Description

  • The Agent installed on an unintended language, how can I change it?

Environment

  • /.

Solution

  • Although there isn't an option to change the Agent language after install via the Configuration interface, you can perform the change via the relevant Agent's ini file.
  • Note that this procedure needs to be done on the device and will require that the Agent not be running to allow editing of the file - so it is not possible to do via a connection to that same Agent:
    1. Open BASupSrvc.ini in notepad;
      • This file can be located under: C:\ProgramData\GetSupportService.
    2. Under [Main] edit the first line to your intended Language:
      • Lang=en for English;
      • Lang=es for Spanish;
      • Lang=fr for French;
      • Lang=pt for Portuguese.
    3. Save the file once edited;
    4. Restart the Take Control Agent service.
