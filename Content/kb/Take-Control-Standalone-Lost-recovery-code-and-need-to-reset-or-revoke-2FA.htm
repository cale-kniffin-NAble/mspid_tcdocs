: Lost recovery code and need to reset or revoke 2FA

Last Modified

Tue Nov 17 11:13 GMT 2020

Description

  • A technician has lost the recovery code required to reset their Two-Factor Authentication setting.
  • An end-user has lost the recovery code required to reset their Two-Factor Authentication setting.
  • I've accidentally enabled the Force Two-Factor Authentication option on a Technician or Main Admin without first setting up Two-Factor Authentication for their profile - how can I disable it?

Environment

  • /

Solution

  • To reset 2FA on an End User, Technician, Admin or Main Admin profile when they no longer recall the recovery code:
    1. Sign in to the Admin Area with your Main Admin account
      • Note that only the Main Admin login can perform these steps.
    2. Go to Management > Technicians and select the intended profile
    3. Under Reset Two-Factor Authentication select Apply.=
    4. When the Operation Completed Successfully message appears, the 2FA setup for that login has been reset.
      • The End User, Technician, Admin or Main Admin will be given the option to setup 2FA again on login.
  • To reset the 2FA of a Main Admin profile that is no longer able to login or should the Main Admin be unavailable:
    • Raise a case with our technical support team.
    • NOTE: It is not possible to revoke the requirement for Two-Factor Authentication usage for the Main Admin or Admin profiles - as these are required to use 2FA/MFA.
  • NOTE: It is only possible to reset 2FA/MFA for an End User or Technician through this method, to disable/revoke 2FA, the Technician/End User must perform the steps outlined in this article: Take Control: Disable 2FA/MFA on Technicians that have previous had it setup
