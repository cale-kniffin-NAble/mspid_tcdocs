- Characters Supportability

Last Modified

Wed Nov 30 21:44 GMT 2016

Description

This article gives information about some characters that will not be displayed by the solutions due to supportability issues.

Environment

All versions

Solution

Solution works with Latin characters. Non-Latin characters represents an issue, but it is expected to have some improvements in the future.

The following languages have unsupported characters:

  • Greek
  • Danish
  • Chinese / Mandarin
  • Korean
  • Arabic
  • Hindi
  • Japanese
  • Thai
