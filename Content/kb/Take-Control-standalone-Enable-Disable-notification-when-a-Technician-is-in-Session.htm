: Enable/Disable notification when a Technician is in Session

Last Modified

Tue Mar 10 16:55 GMT 2020

Description

  • How to notify the Remote User when a Technician is performing a remote session.
  • How to stop the remote session notification from showing.

Environment

  • /
    • Take Control Agent
  • Windows OS
  • macOS

Solution

  • Windows OS
    1. Login to the Remote Device;
    2. Restore the Take Control Server Configuration application;
      • It can be searched for but it's usually found in the System Tray to the bottom right corner of the Desktop;
    3. Press the Advanced Setup button (cogwheel icon);
      • Should it be grayed out, press the Enter Administration Mode button on the top right corner to enable Admin Mode;
        • You'll need Administrative Privileges to proceed so a pop up window will appear informing about this;
    4. Select Security;
    5. Enable/Disable the option Notify when a tech is in session, as desired;
    6. Press Apply.
    • To do this from the Admin Area without having to access the device directly please check the article [%=General.ProductName_tc-sa%]: How to change Agent settings
    • To do this from on a Take Control macOS agent please check the article above as well.
