Take Control (All Versions): Abnormal Program Termination

Last Modified

Wed Sep 16 10:05 GMT 2020

Description

  • When launching the , the following error occurs: "Error: Abnormal Program Termination";
  • In the context of the Remote Background feature also prompts the same error because it shares the Viewer protocol and components;

Environment

  • /
    • Take Control

Solution

  • This error commonly occurs when a MSXML Parser component is corrupted or missing on the local device;
  • Open an elevated command line and run the command: sfc /scannow - in order to scan and repair any Windows components;
  • If this does not fix the issue, please install Microsoft XML Parser component: MSXML 4.0 Service Pack 3 (Microsoft XML Core Services);
    • Note that if installing MSXML 4.0 corrects the issue - this confirms that the MSXML parser included in your Windows OS is damaged/unstable.
    • Additionally note that MSXML 4.0's version is outdated and unsupported by Microsoft which can lead to further roadblocks in the future, so it's strongly suggested to repair or reinstall the afflicted Windows OS to repair/install the missing files.
