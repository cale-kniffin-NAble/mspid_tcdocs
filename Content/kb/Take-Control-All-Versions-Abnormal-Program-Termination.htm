Take Control (All Versions): Abnormal Program Termination

Last Modified

Wed Aug 4 10:05 GMT 2021

Description

  • When launching the , the following error occurs: "Error: Abnormal Program Termination";
  • In the context of the Remote Background feature also prompts the same error because it shares the Viewer protocol and components;

Environment

  • /
    • Take Control

Solution

  • This error commonly occurs when a MSXML Parser component is corrupted or missing on the local device;
  • Open an elevated command line and run the command: sfc /scannow - in order to scan and repair any Windows components;

If this does not resolve the issue, please install the Microsoft XML Parser components for your Operating System.

  • List of Microsoft XML parser (MSXML) versions: https://docs.microsoft.com/en-us/previous-versions/troubleshoot/msxml/list-of-xml-parser-versions

  • Microsoft Download Center (focussed on MSXML): https://www.microsoft.com/en-us/download/search.aspx?q=msxml

Note

  • If installing the MSXML components corrects the issue - this confirms that the MSXML parser included in your Windows OS is damaged/unstable.

  • Although we have received report of MSXML 4.0 Service Pack 3 (Microsoft XML Core Services) alleviating the issue, please be aware that installing an outdated and unsupported by Microsoft version of the MSXML components can lead to further roadblocks in the future. As such it is strongly suggested to repair or reinstall the afflicted Windows OS to repair/install the missing files.
