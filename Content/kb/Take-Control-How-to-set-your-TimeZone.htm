Take Control: How to set your Time Zone

Last Modified

Thu Jan 31 17:42 GMT 2019

Description

  • How to setup our timezone for Take Control.

Environment

  • (

Solution

  • You can set the company default as follows:
    1. Login to your Admin Area;
    2. Go to Profile>Identification;
    3. Select the BASIC INFORMATION tab;
    4. Select the intended option in the Timezone dropdown menu;
    5. Click the Save button (bottom right corner).
  • You can also set the Timezone per Technician (which will override the company timezone setting when they're logged into the Admin Area) as follows:
    1. Login to your Admin Area;
    2. Go to Management>Technicians;
    3. Select their Technician's Profile;
    4. Set the Time Zone field as intended;
    5. Click the Save button (bottom right corner).
