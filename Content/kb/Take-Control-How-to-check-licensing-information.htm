Take Control: How to check licensing information

Last Modified

Wed Feb 12 12:51 GMT 2020

Description

  • How to check licensing information?
  • How many concurrent licenses are at place?
  • How to check renewal date?

Environment

  • /

Solution

  1. Login to the Admin Area;
  2. On the top right corner, click the User Menu;
  3. Select the option Licensing Information;
    • Alternatively, you may access the Licensing Information page via Management > Licenses;
  4. Check the licensing details listed under the DETAILS tab;
  • Only technicians with administrator privileges will be able to access this information!
