: How to start a VoIP Call

Last Modified

Tue Sep 26 21:25 GMT 2017

Description

  • How to initiate a VoIP call during an session

Environment

Solution

  1. Start your remote session
  2. Select the Session menu (bottom of the viewer window, left of center)
  3. Select the StartVoIP Call option (or use the Shift+Ctrl+F2 hotkey)
  4. A VoIP call should commence using the default audio devices on both the local and remote devices
