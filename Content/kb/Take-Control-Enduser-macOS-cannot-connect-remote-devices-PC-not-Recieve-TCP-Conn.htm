Enduser on macOS cannot connect to remote devices - PC is not Ready to Recieve TCP Connections

Last Modified

Wed Dec 06 14:53 GMT 2017

Description

  • Mac clients suddenly cannot connect to remote devices with agent v.6.80.20 through enduser viewer
  • Error: "PC is not ready to recieve TCP connections"

Environment

  • macOS
  • Enduser viewer v.1.00.12
  • Agent version v.6.80.20

Solution

  • Update the end user viewer
    1. Go to the end user area
    2. Insert the end user credentials
    3. Click the button Install extension
    4. Run the installer
