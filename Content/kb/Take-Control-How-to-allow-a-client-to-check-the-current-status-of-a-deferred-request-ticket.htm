Take Control: How to allow a client to check the current status of a deferred request (ticket)

Last Modified

Tue Mar 26 14:48 GMT 2019

Description

  • How to allow a client to check the current status of a deferred request (ticket).

Environment

Solution

  • To allow the remote end to check the status of their tickets is something to be enabled by individual device, as follow:
  1. Login to the Admin Area;
  2. Navigate to Devices > select the intended device;
  3. On the GENERAL tab, selected by default, set to on the option Show History Tab on "Create a Deferred Support Request" page;
  4. Save changes;
  • When the remote end right clicks the agent sis tray icon and selects the Support tickets option. they will have access to their own tickets history.
