Take Control: How to define an out of work schedule warning for attended support Applets

Last Modified

Mon Jul 22 15:24 GMT 2019

Description

  • How to define an out of work schedule warning when launching an applet.

Environment

Solution

  1. Login to your Admin Area;
  2. Navigate to Profile > Applet Setup;
  3. Select the WARNINGS tab;
  4. Click on Add New or on the Blue Square in the Calendar representing an existing Warning (should you wish to edit rather than create);
    • The schedule set for the Week Profile should define the out of work hours;
  5. Fill the form as intended (be sure to include the URL, Name and ensure that the End Time is set later than the Start Time);
  6. Click the Save button.

Article Number: 133022

Internal notes: Check article 000063391 on how to choose a URL to display to the customer when the Applet is closed.
