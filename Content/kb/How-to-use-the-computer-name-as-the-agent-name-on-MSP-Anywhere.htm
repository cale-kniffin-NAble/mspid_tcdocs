Take Control: Set Agent installs to use the device's actual name

Last Modified

Tue Apr 21 11:55 GMT 2020

Description

  • How to set the Take Control Agent install to use the Device's Actual name.

Environment

  • /.

Solution

  1. Login to your Admin Area;
  2. Navigate to Profile > Agent Setup;
  3. Under GENERAL, enable the Use the Computer Name as the Agent Name option.
    • This option when enabled will make Agent installers use the device's actual name/network name as it's Agent name, as opposed to the name requested for the install. If disabled, the install will use the name you provide instead.
  4. Save when done (bottom right corner).
  • Note that these settings won't apply to any existing Agents (although their names can be changed manually via the right click > rename option in the Console or the Edit option in the Admin Area's Devices list) but it will apply to all future installs.
