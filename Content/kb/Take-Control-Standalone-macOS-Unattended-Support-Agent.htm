: macOS Unattended Support (Agent)

Last Modified

Wed Sep 09 13:34 GMT 2020

Description

  • How to generate a Take Control Agent installer for a macOS device?

Environment

  • /
  • macOS.

Solution

  • via the Take Control Tech Console:
  1. Login to your Technician's Console application;
  2. Select the MY COMPUTERS tab;
  3. Left-click on the ADD COMPUTER button neat the bottom-left corner;
  4. Fill out the form to the right of the window;
    • Note that although only the Computer Name field is mandatory, you should ensure that the Device Type is set as MacOS 10.10+.
  5. Left-click on the GENERATE LINK button;
    • Each MacOS install requires it's own PIN Code generated from the Add Computer process - please save the PIN Code generated at this step.
  6. Provide your client with the generated PIN code and have them download the macOSAgent installer from www.startcontrol.com/, under the MAC tab;
    • Alternatively, provide them with the following link: https://swi-rc.cdn-sw.net/remote_control/mac/sw_remote_control_agent_installer.zip.
  7. Run (or have your client run) the agent installer on the intended macOS device;
  8. Input the previously saved PINCode when prompted and complete the installation;
  • via Admin Area:
    • Provided Admin Area Access is granted!
  1. Login to the Admin Area;
  2. Enable Remote Control Mode (on the top right side, the dot with two curved lines icon);
  3. Navigate to Devices;
  4. Click the + Add device button;
  5. Fill the form as aforementioned;
    • Do not forget to change the Operating System field to MacOS 10.10+;
  6. Click Save;
  7. Provide your client with the generated PIN code and have them download the macOSAgent installer from www.startcontrol.com/, under the MAC tab;
    • Alternatively, provide them with the following link: https://swi-rc.cdn-sw.net/remote_control/mac/sw_remote_control_agent_installer.zip.
  8. Run (or have your client run) the agent installer on the intended macOS device;
  9. Input the previously saved PINCode when prompted and complete the installation;
  • Once the Agent is installed and communicating successfully, you should be able to remote access the newly added macOS device!
