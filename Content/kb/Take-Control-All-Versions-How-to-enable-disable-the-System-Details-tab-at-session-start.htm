Take Control (All Versions): How to enable/disable the System Details tab at session start

Last Modified

Thu Mar 28 15:23 GMT 2019

Description

  • How can you enable or disable the System Details tab from popping up at the session start?

Environment

  • /

Solution

  • If you wish to enable or disable the System Details tab from popping up at every session start, you should:
    1. Start a new Take Control Session
    2. Click on the three vertical dots at the top right corner of the
    3. Check/Uncheck the box System Details tab at a Session start
