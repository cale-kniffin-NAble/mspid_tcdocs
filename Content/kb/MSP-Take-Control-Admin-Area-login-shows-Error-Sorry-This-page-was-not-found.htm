: Admin Area login shows Error: "Sorry. This page was not found."

Last Modified

Thu Sep 20 16:54 GMT 2018

Description

  • Attempting to login to the Admin Area, I input my username (e-mail) and then press Login, but I'm directed to an "sso_error.php" error page with the following error:
    • Error: "Sorry. This page was not found."

Environment

Solution

  • This error page (https://admin.corporate.beanywhere.com/admin_area_/sso_error.php) is loaded when you're attempting to login to a account that is different from your currently active SSO login on this browser. Please Logout of your account before attempting to login to a new one:
    1. Navigate to https://sso.navigatorlogin.com/ and click Logout to ensure you're not logged into as an unintended account;
    2. Navigate to the login page
    3. Attempt to login with your username (e-mail), this should re-direct you to the login page;
    4. Make sure that when logging into , you're logging in with the SSO account with the same e-mail address used in step 3;
    5. You should now be logged into the admin area.
  • If you are looking to run two simultaneous logins for , for example, please use an Incognito Mode browser window for the second login.
