Take Control: How to change technician's name

Last Modified

Tue Sep 03 12:51 GMT 2019

Description

  • How to change a Technician's name within Take Control/?

Environment

  • /

Solution

  1. Login to the Admin Area;
    • This procedure must be done by the technician who is looking to change their name.
  2. Click on the User Menu, located on the top right corner;
  3. Choose the option Edit your Details;
  4. Change the First name and/or Last name fields as desired;
  5. Type your password on the Password field to validate the changes;
  6. Click the button SAVE DETAILS.
