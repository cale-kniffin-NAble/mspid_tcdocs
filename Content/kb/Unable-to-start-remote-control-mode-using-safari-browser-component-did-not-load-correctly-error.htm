Unable to start remote control mode using safari browser, its showing an error "The component did not load correctly. Please make sure the cookies from the following domains are allowed: widget-swi-rc.com, navigator.sso.com, admin.swi-tc.com"

Last Modified

Thu Oct 22 11:14 GMT 2020

Description

  • When starting remote control mode in the admin area using safari browser, its showing an error "The component did not load correctly. Please make sure the cookies from the following domains are allowed: widget-swi-rc.com, navigator.sso.com, admin.swi-tc.com"

Environment

Solution

  • Ensure that the following domains are not blocked on the device: widget-swi-rc.com, navigator.sso.com, admin.swi-tc.com
  • Check browser settings:
    1. Open Safari browser
    2. Go to Preference
    3. Click Privacy 
    4. On the Website tracking, uncheck Prevent cross-site tracking 
    5. Reload the browser and try again
