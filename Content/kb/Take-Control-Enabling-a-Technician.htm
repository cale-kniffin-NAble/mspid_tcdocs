: Enabling a Technician

Last Modified

Wed Feb 26 10:18 GMT 2020

Description

  • How to enable a Technician's Profile so they can regain access to Take Control
  • The Main Administrator's profile cannot be disabled.
  • How to show only disabled technicians?

Environment

  • /

Solution

  1. Login to your Admin Area;
  2. Navigate to Management > Technicians;
  3. Add a filter to show all disabled technicians by clicking the filter icon next to the search box and selecting the Disabled checkbox;
  4. Use the search box to find the desired Technician's Profile;
  5. Click the Enable option to turn it on and enable the technician;
  6. Click the Save button.
