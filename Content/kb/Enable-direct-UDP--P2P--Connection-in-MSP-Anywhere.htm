Enable direct UDP (P2P) Connection in

Last Modified

Wed Nov 30 21:44 GMT 2016

Description

This article reviews the option to connect over the applet directly to the end user with a UDP connection. This will bypass the gateway and do a direct peer to peer connection, which may increase performance and help in troubleshooting connection speed issues.

Environment

  • Technician console v6.77.12 and above (Applet connections only)

Solution

:

  1. Open the technician console and login. Take note of the version (you can update by logging into the Admin Area and selecting the Console link on the left hand menu).


  2. Go to the Options menu and select Try UDP Connections First


Note: From now on whenever your end user starts the applet (pincode/link), it will try and bypass the gateway and try a direct UDP connection first.
