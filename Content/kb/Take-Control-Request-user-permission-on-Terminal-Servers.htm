Take Control: Request User Permission on Terminal Servers

Last Modified

Tue 12 Jan 11:06 GMT 2021

Description

  • The option "Request User Permission" is set on the Take Control Agent for a Terminal Server but no pop up is showing to the remote users when attempting to connect.

Environment

  • /
    • Take Control

Solution

  • For Take Control Agents installed on Terminal Servers the "Request User Permission" pop up will only appear to the local server user, meaning it will only appear on the console session.
  • If the users are, for instance, connected through RDP sessions, the "Request User Permission" pop up, is not going to be displayed.
