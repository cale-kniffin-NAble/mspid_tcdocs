: Command Line Switches and Parameters to uninstall the Take Control Agent

Last Modified

Mon Dec 28 17:21 GMT 2020

Description

  • This article instructs on how to silently uninstall the Take Control Agent.

Environment

  • /

Solution

  • Run the uninstall executable located under C:\Program Files (x86)\BeAnywhere Support Express\GetSupportService, with the following parameters:
    • uninstall.exe /S /R
      • /S will force a silent uninstall (no user interaction) and /R will refresh registry changes after the uninstall completes.
  • For information about how to silently install the agent, see[%=General.ProductName_tc-sa%]: Agent Silent Install.
