Take Control (All Versions): How to fix outdated Root Certificates

Last Modified

Mon Oct 21 11:43 GMT 2019

Description

  • How to fix Take Control outdated Root Certificates.
  • How to fix invalidated Certificate Revocation List (CRL).

Environment

  • /
  • Take Control

Solution

  • Verify that your client has the most recent update for their root certificates:
    • Trusted root certificates that are required by Windows Server 2008 R2, by Windows 7, by Windows Server 2008, by Windows Vista, by Windows Server 2003, by Windows XP, and by Windows 2000
    • Support for urgent Trusted Root updates for Windows Root Certificate Program in Windows
  • In the case of Firewall issues blocking the validation of the certificates the partner should allow traffic according to the information published by the issuer of the certificate. For instance:
