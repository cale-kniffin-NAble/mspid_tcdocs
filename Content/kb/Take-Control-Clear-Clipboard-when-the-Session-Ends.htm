Take Control: Clear Clipboard when the Session Ends

Last Modified

Fri Dec 13 17:59 GMT 2019

Description

  • Instructions on how to automatically clear the clipboard at the end of each remote session.

Environment

  • /
    • Take Control

Solution

  • This setting has to be setup individually, per ;
  • To configure this, please follow the instructions below:
    • While on a remote session:
    1. Click the Remote Desktop tab;
    2. Hover over Interactions > Clear Clipboard when the Session Ends;
    3. Select the desired option between the following:
      • Disabled: Does not clear the clipboard;
      • Local Only: Clears the local clipboard (tech side);
      • Remote Only: Clears the Remote Clipboard (client side);
      • Local and Remote: Clears both clipboards (tech and client);
  • The setting chosen will be applied to all remote sessions launched from the local device at hand;
