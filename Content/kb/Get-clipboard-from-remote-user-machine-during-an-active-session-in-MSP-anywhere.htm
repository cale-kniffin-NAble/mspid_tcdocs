Get clipboard from remote user machine during an active session in

Last Modified

Wed Nov 30 21:44 GMT 2016

Description

The clipboard is a short-term storage area for all kinds of information that you have copied from one place and you plan to use in another place. You can copy just about anything to the clipboard - strings of text, images, video, files, folders, shortcuts, etc.

This article provides steps to get clipboard from remote user's machine during an active session in .

Environment

  • All versions

Solution

  1. Establish a connection to the remote device using the console, or through in .
  2. Once successful, click on the Interactions tab at the lower right hand corner of the current active remote control session.
  3. Select 'Copy/Paste', then click on 'Get Remote Clipboard'.
