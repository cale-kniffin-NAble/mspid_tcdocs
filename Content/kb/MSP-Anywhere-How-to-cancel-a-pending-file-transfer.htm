: How to cancel a pending file transfer?

Last Modified

Fri Mar 30 15:32 GMT 2018

Description

  • How to cancel a pending file transfer on an session?
  • Are there consequences for cancelling an ongoing transfer?

Environment

Solution

  1. Select the File Transfer tab;
  2. Under FILE TRANSFER QUEUE, right click on the chosen file;
  3. Select either:
    • Cancel Selected: Cancel transfer of chosen file;
    • Cancel All: Cancel all pending and ongoing transfers;
  • Please note that cancelling an ongoing transfer will leave the already transferred remnants of the file on the receiving end, these will need to be manually deleted.
