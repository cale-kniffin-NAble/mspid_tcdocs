Remote Support Session Options

Last Modified

Wed May 01 17:25 GMT 2019

Description

  • What are the Take Control remote support session options to install on client devices?

Environment

    • Applet
    • Agent

Solution

  • There are two types of remote support sessions available through : Applet and Agent
  • Applet sessions allows remote support with no additional application installed on the end user's machine.
    • More information on Applet sessions can be found in the User's Guide here.
  • Agent sessions require an application to be installed on the end user's machine but allows for unattended sessions as well as more advanced settings to be applied.
    • More information on Agent sessions can be found in the User's Guide here.
