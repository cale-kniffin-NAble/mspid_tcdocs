Take Control (All Versions): How to Enable/Disable the Viewer keyboard shortcuts?

Last Modified

Tue Aug 27 14:43 GMT 2019

Description

  • How to Enable/Disable the keyboard shortcuts during a Take Control remote support session?

Environment

  • /;
  • :
    • Take Control.

Solution

  • Windows:
    • To Enable/Disable the keyboard shortcuts, while the Viewer is open, press the Scroll Lock key.
    • On the top right corner of the Viewer window you'll see a visual Scroll Lock indicator (Square with an Arrow pointing down) with two possible states:
      • Scroll lock Off- Viewer shortcuts are ENABLED (Grey)
      • Scroll lock On - Viewer shortcuts are DISABLED (White)
  • macOS
    • To disable/enable the Viewer keyboard shortcuts, while the Viewer is open, use the key combination cmd + option + 0 (zero)

Internal notes: The Viewer includes several internal keyboard shortcuts:.If the Scroll key is Off - Keyboard shortcuts recognized by the viewer will be executed locally and will not be passed to the remote computer..If the Scroll key in On - All keyboard shortcuts will be passed to the remote computer. No shortcuts will be executed by the Viewer
