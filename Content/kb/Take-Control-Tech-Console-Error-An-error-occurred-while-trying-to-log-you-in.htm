Take Control Tech Console Error: "An error occurred while trying to log you in"

Last Modified

Mon Aug 31 12:36 GMT 2020

Description

  • Failing to login on the Take Control Tech Console Application.
  • Error: " An error occurred while trying to log you in. Please try again. If the problem persists please contact us. Thank you!".

Environment

  • /
  • Windows

Solution

  • This error message can occur due to a missing or damaged Microsoft MSXML Parser component
  • Please download and install the following: Microsoft Core XML Services (MSXML) 6.0 Service Pack 1

Article Number: 133901
