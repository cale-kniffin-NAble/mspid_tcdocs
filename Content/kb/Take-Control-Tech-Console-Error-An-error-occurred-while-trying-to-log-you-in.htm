Take Control Tech Console Error: "An error occurred while trying to log you in"

Last Modified

Mon 01 Mar 13:33 GMT 2021

Description

  • Failing to login on the Take Control Tech Console Application.
  • Error: " An error occurred while trying to log you in. Please try again. If the problem persists please contact us. Thank you!".

Environment

  • /
  • Windows

Solution

  • This error message can occur due to a missing or damaged Microsoft MSXML Parser component.
  1. Download and install the following: Microsoft Core XML Services (MSXML) 6.0 Service Pack 1
    • A restart may be necessary to resolve this after installing the XML Service Pack.
  2. If this does not resolve the issue, sign out of the Take Control Console by selecting the Log Off button on the top right of the Console, then sign back in.
  3. If this does not resolve the issue, uninstall the Take Control Console from Add/Remove Programs, then download and reinstall the Console.
  4. If this does not resolve the issue, please contact Technical Support for further assistance.
