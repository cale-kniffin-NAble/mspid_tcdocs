: How to Install Multiple Instances/Agents on the same Computer

Last Modified

Fri Mar 27 12:23 GMT 2020

Description

  • How to install more than one Take Control Agent on the same workstation?
  • Attempting to install the Take Control Agent, but it does not show up in my Device list after the install - another company's Agent is already on the device;
  • Installed Take Control Agent but it shows another company's logo, it also does not appear on My Computers list.

Environment

  • /;
  • Windows.

Solution

  1. Login on the Tech Console;
  2. Press the My Computers tab;
  3. Press the Add Computer button;
  4. Fill out the form on the right side of the Console;
    • Only the "Device Name" field is mandatory - although it's recommended that you review all fields before Generating link (such as those relating to the Operating System and expiration dates).
  5. Press the Generate Link button;
  6. Access the link and download the Agent Installer (default labelled as BAManagement.exe);
  7. Run the Agent installer on the remote device with the following syntax: BAManagement.exe /Instance InstanceName

    • Replace the InstanceName string to your desired Instance Name - keep in mind that it should be under 16 characters and use only alphanumeric characters (no special symbols, spaces or punctuation).
