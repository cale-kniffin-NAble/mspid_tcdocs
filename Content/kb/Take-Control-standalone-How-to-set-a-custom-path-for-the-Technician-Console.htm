: How to set a custom path for the Technician Console

Last Modified

Mon Apr 27 10:32 GMT 2020

Description

  • Instructions on how to change the default Technician Console installation path.

Environment

  • /

Solution

  1. Download the Take Control console installer on the target device;
    1. Login to the Admin Area;
    2. Go to Downloads;
    3. Download the Technician Console;
  2. Run the Console installer through the Windows Command Prompt, as follows:
  • /D=<custom path>
    • Replace <custom path> with the intended install path.
