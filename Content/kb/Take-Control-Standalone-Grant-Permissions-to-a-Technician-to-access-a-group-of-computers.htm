: Grant Permissions to a Technician to access a group of computers

Last Modified

Wed Aug 19 14:24 GMT 2020

Description

  • Allowing a Technician to access a specific group of computers in their Take Control Tech Console and Admin Area webpage.

Environment

  • /

Solution

  • By Technician:
  1. Login to the Admin Area;
    • Please note that you will need sufficient permissions to access the admin area, if you are not an admin - please reach out to one of the account's administrator;
  2. Click on Management;
  3. Click on Technicians;
  4. Select the Technician;
  5. Click on PERMISSIONS;
  6. On the GLOBAL tab ensure that under My Computers Management the option Access to My Computers is enabled;
  7. Click on Select Groups and select all groups you want to grant access to the technician;
    • Note that the technician will always have access to the Root group and, consequentially, any Agents within it;
  8. Click OK;
  9. Click Save;
  10. The Technician will need to login to the Console to pick up the new settings.
  • By Agent Group:
    1. Login to the Admin Area;
    2. Navigate to Devices;
    3. Edit the intended group by clicking the vertical ellipsis symbol next to the group;
    4. Under the Permissions section, click the Technicians drop-down to chose the desired technicians to access the target group
    5. Save when done;
      • Note that the technician needs to have sufficient permissions to access the My Computers tab;
    6. The Technician will need to login to the Console to pick up the new settings.
