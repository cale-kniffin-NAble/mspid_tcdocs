Does Take Control support Linux devices without a monitor for Remote access?

Last Modified

Mon Aug 19 04:39 GMT 2019

Description

  • There are Linux devices without an actual monitor in the environment.

Environment

Solution

  • This is supported if Xorg and Desktop Environment is properly configured. Click HERE for the installation guide.
  • If Remote Desktop Environment is not installed at all, it is still possible to use features like Remote Shell and other supported features.
