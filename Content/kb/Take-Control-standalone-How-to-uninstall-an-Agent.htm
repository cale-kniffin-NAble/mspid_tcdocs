: How to uninstall an Agent

Last Modified

Mon Dec 28 17:32 GMT 2020

Description

  • This article covers instructions on how to uninstall the Take Control agent.

Environment

  • /;

Solution

  • From Tech Console
    1. Login into your Technician Console application;
    2. Open MY COMPUTERS tab;
    3. Right-click intended device;
    4. Select Delete (or use the CTRL+D hotkey combination);
      • A Command will be sent to these devices to trigger the agent uninstall, this may take a few minutes;
      • If a device is offline, the uninstall command should proceed as soon as the device comes back online.
  • From Admin Area:
    1. Login to your Admin Area;
    2. Select Devices menu (left pane);
      • Filter listed results by choosing groups and using Filters sub menu;
    3. Select intended devices by ticking the checkbox to left of their name;
    4. Click More on the top pane;
    5. Select Delete;
      • A Command will be sent to these devices to trigger the agent uninstall, this may take a few minutes;
      • If a device is offline, the uninstall command should proceed as soon as the device comes back online.
