: End User Portal through Android or iOS Systems

Last Modified

Fri Mar 22 09:50 GMT 2019

Description

  • Can an End User access the End User Portal through Android or iOS devices?

Environment

  • /

Solution

  • The plug-in used in the End User Portal is only compatible with Windows and macOS systems. End Users can't access the End User Portal through Android or iOS devices
