Take Control: How to create secrets vaults

Last Modified

Mon Oct 22 15:43 GMT 2018

Description

  • Steps on how to create Secrets Vaults

Environment

Solution

  1. Login on your Admin Area;
  2. Navigate to Management > Secrets Vaults;
  3. Click the Add button on the Vault area (top right side);
  4. Fill the Vault Name and Vault Description fields;
  5. Click the Continue button;
  6. Set a Vault Password and click Continue;
  7. Save the recovery code in a safe place
    • Use the recovery code to re-issue Passwords for any Technician with access to this Vault;
    • You will not be able to recover the access to the Vault contents should you lose this code and your Vault Password;
  8. Click Ok.
