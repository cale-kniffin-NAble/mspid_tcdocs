How to generate an Exclusive Link or Applet direct download link on

Last Modified

Tue Apr 21 13:22 GMT 2020

Description

  • How to generate an exclusive link on Take Control.
  • How to create a button for clients to start a remote session with Take Control.
  • How to provide applets without having to generate PIN codes.

Environment

  • /
  • Windows

Solution

  1. Login to your Admin Area;
  2. Go on Profile > Integration;
  3. Select the EXCLUSIVE LINK tab;
  4. Choose the startcontrol.com domain and set the intended url;
  5. Click Save (lower right corner) when done.
  • Exclusive Link: Personalized link pointing to StartControl;
  • Applet direct link: Directly downloads an applet without the need of a PIN code;
    • The applet direct link only works for Windows OS.
