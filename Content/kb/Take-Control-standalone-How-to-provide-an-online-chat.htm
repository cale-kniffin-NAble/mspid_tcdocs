: How to provide an online chat

Last Modified

Tue 12 Jan 10:35 GMT 2021

Description

  • How to generate a Web Chat session through Take Control.
  • How to integrate the Take Control web chat link on your webpage.

Environment

  • /

Solution

  1. Navigate to your Admin Area;
  2. Go to Management > Technicians>Select the desired tech;
  3. Go to PERMISSIONS > CONSOLE FEATURES;
  4. Ensure the Chat with customers option is set to ON;
  5. Click the Save button;
  6. Click the Copy link button;
  7. Paste the URL on your desired webpage within an href attribute (or similar);
  8. The webpage should now feature the web chat option.
    • Ensure that the intended Technician (or Technician/s within the intended department) are online in their Console's Chat Window in order to be able to respond to chat requests
