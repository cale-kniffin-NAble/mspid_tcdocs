How to enable a user to remote into their computers without them requiring someone at machine to accept remote session

Last Modified

Wed Aug 05 08:59 GMT 2020

Description

  • How to enable a user to remote into their computers without them requiring someone at machine to accept remote session

Environment

Solution

  1. Click Settings
  2. Select Take Control
  3. Click Policy
  4. Click on policy you wish to edit
  5. Click Edit
  6. Click on Security
  7. Find option Request user permission
  8. Disable option
  9. Click Save
