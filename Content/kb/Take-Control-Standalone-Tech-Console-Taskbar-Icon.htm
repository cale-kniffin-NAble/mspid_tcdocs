: Tech Console Taskbar Icon

Last Modified

Fri May 31 11:10 GMT 2019

Description

  • How can we create an Take Control Tech Console icon on the Windows Taskbar?

Environment

  • /

Solution

  • If you wish to create an Take Control Tech Console icon on your Windows Taskbar, you should:
  1. Right click on the icon present on your Desktop
  2. Select Pin to Taskbar
  • You can also:
  1. Go to C:\Users\<User Name>\AppData\Local\BeAnywhere Support Express\Console
  2. Right-click on BAConsoleApp.exe
  3. Select Pin to Taskbar
