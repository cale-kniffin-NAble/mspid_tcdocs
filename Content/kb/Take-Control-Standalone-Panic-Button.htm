: Panic Button

Last Modified

Thu Jul 11 08:53 GMT 2019

Description

  • I need to temporarily increase my number of concurrent techs, how can I do this?
  • My technicians can't login and I've hit my license limit, can I do anything to bypass this temporarily?

Environment

  • /.

Solution

  • Selecting the panic button will increase the license limit to 999 for 10 hours and will allow as many technicians to be logged in concurrently (up to 999) as needed.
  • This is useful for critical situations where additional manpower is needed for a short period of time, thus defeating the need to use other tools or buy additional licenses.
  • Note that uses of the Panic Button are limited, be sure to review how many hits you have left before activating the Panic button.
  • It can be activated on the Admin Area as follows - although only as the Main Admin:
    1. Click the User Menu, located on the top right corner;
    2. Select the Licensing Information menu;
    3. Select the Advanced tab;
    4. Click the Panic button;
    5. You will be prompted to confirm this action, please read the prompt carefully as it states how many uses remain.
