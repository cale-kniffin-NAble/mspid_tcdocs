Session Video Recordings Default Settings

Last Modified

Mon Dec 28 17:49 GMT 2020

Description

  • Configure the default settings for session video recordings for a specific technician.
  • How to allow a technician to record remote sessions?

Environment

Solution

  • By default, videos recordings are stored here:
    • Windows: C:\Users\Bruce\AppData\Local\BeAnywhere Support Express\Console\FLV (C:\Users\Username\AppData\Local\BeAnywhere Support Express\Console\FLV
    • macOS:/Users/<user>/Documents/MSP Anywhere Files/Console/FLV
  • Default storage location can't be changed, but the storage location of a recording can be changed within a remote session, as follows:
    1. Open a remote session;
    2. Click the Preferences menu (vertical ellipsis icon on the top right corner);
    3. Select Video Storage Location > Custom;
    4. Choose the desired folder;
    5. Click OK;
  • By default, technicians will have video recording disabled. This can be changed by following the steps below:
    1. In the Admin Area, navigate to Management > Technicians and select the desired technician from the list;
    2. Click on the DEFAULT SETTINGS tab;
    3. Under the "Define the video recording settings for this Technician" section choose one of the following options:
      • Disabled - This setting disables video recording for the technician's remote sessions. Session recording can't be enabled by the technician while in a remote session;
      • No, by default - This setting disables video recording for the technician's remote sessions, but it can be enabled by the technician while in a remote session;
      • Yes, by default - This setting enables video recording for the technician's remote sessions, but it can be turned off by the technician while in a remote session;
      • Yes, always - This setting enables mandatory video recording for the technician's remote sessions. The technician cannot turn it off;
    4. Click Save.
