: Setup branding in the Admin Area

Last Modified

Wed Dec 30 10:41 GMT 2020

Description

  • How to setup your own company logo through the Take Control Admin Area
    • This logo will appear in your custom "startcontrol.com" page and on any installed Take Control Agents (resident application installed on a client's device)

Environment

Solution

  1. Login to your Admin Area;
  2. Navigate to Profile > Identification;
  3. Select the BASIC INFORMATION tab;
  4. Press the Choose Image button;
  5. This will open a system window (explorer in Windows) allowing you to choose a file;
    • The image file must be in JPEG/JPG format and equal to or under 200 KB in size;
  6. Once the upload completes, you should now have a new logo;
  7. Save when done.
