Take Control: Getting the error "calling card prefix is undefined" when generating a calling card

Last Modified

Wed May 15 10:44 GMT 2019

Description

  • Unable to create a calling card.
  • Getting the error "calling card prefix is undefined" when generating a calling card.

Environment

Solution

  • This error occurs when a prefix is not defined for the calling card, like the error states.
  • Please ensure a prefix is set for the calling card as follows:
  1. Login to the Admin Area;
  2. Navigate to Profile > Identification;
  3. Select the PIN/CALLING CARDS tab;
  4. Under the field Calling Cards Prefix set the intended prefix;
    • The prefix identifies the Calling Cards as yours and can help reinforce your company's brand. It is composed of 4 to 8 digits - letters A-Z (case sensitive) or numbers 0-9.
  5. Click Save.
