: Agent reverts button from Cancel Request to Request Assistance after some time while the Request is still pending

Last Modified

Wed Apr 11 12:33 GMT 2018

Description

  • After a client presses the Request Assistance button on an Agent and some time has passed, the button reverts from Cancel Request back to Request Assistance;
  • The issue is still pending on my console, but the button appears as "Request Assistance" on the client's end;

Environment

  • Windows

Solution

  • After approximately 150 seconds, the Agent defaults it's interface to display the Request Assistance button - even if a request is still pending.
  • This is done by design to prevent the Agent from perpetually displaying a Pending Request, as requests submitted from the Agent do not expire (until accepted or rejected by a Technician).
