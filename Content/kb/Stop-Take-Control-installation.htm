Stop Take Control installation

Last Modified

Tue Jan 29 19:49 GMT 2019

Description

This article provides steps to prevent Take Control from launching during boot up.

Environment

version 10.2 and later

Solution

Note: You cannot stop the Take Control installation, you just have to find the best way to prevent BASupSrvcCnfg.exe " from automatically launching with the user boot start.

Do any of the following:

  • Delete the registry entry in:

Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Run

  • In Run, right-click and select Export (this way you keep a registry key backup). Delete the following:

BASupSrvcCnfg.exe

  • If the partner is an partner too, he can also create a Policy to prevent the process from running on Start up.


Note: After these changes, you will not be able to use the chat feature and the remote user will not have the ability to Request Support.

If you are referring to instead of Take Control, you will not be able to stop the installation, but you can use the instructions provided above. The registry path can be a bit different from Take Control to .
