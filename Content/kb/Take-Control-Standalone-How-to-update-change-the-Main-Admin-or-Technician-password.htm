: How to update/change the Main Admin or Technician password

Last Modified

Thu Jul 18 18:40 GMT 2019

Description

  • How to update main admin password in Take Control?
  • How to change the password of the Take Control main admin?
  • How to change the password of the Take Control technician?

Environment

  • /

Solution

  • Update password via Technician Console application:
    1. Login to your Take Control Tech console;
    2. Click the Setting menu (cogwheel icon on the top right corner);
    3. Click the Change Password option;
    4. You'll be directed to the relevant Password Change page;
  • Update password via the Take Control admin area:
    1. Login on your Admin Area;
    2. Select the User Menu on the top right corner;
    3. Select the Change Password option;
    4. You'll be directed to the relevant Password Change page.
