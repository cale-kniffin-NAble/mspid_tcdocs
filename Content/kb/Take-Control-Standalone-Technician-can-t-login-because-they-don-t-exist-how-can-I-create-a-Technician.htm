: Technician can't login because they don't exist, how can I create a Technician?

Last Modified

Thu Aug 20 14:38 GMT 2020

Description

  • How to create a Take Control Technician?
  • Technician can't login because they don't exist in Take Control yet.

Environment

  • /.

Solution

  1. Log in to the Admin Area
  2. Navigate to Management > Technicians
  3. On the Technicians tab, click on Add button
  4. Enter the Name, Username (Email) and Expiration Date (optional) on the newly presented window
  5. Save using the Save Button on the bottom right corner of the window
  • For more information, see Create and manage technicians.
