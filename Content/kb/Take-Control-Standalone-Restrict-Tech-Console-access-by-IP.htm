: Restrict Tech Console access by IP

Last Modified

Tue Jan 28 16:05 GMT 2020

Description

  • How to restrict access to Take Control Tech Console by IP?
  • How to restrict technicians to login exclusively from certain computers?

Environment

  • /

Solution

  1. Go to the Admin Area;
  2. Navigate to Profile > Identification;
  3. Select the SECURITY tab;
  4. Limit the access to both the Admin Area and the Technician Console by authorizing only specific sets of IP addresses, as intended;
  5. Click the Save button.
