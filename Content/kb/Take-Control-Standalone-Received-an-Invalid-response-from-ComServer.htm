Take Control Standalone: Received an Invalid response from ComServer

Last Modified

Mon Aug 31 12:37 GMT 2020

Description

  • When trying to launch a Take Control session from the Console or from the End User area, an error is received:
    • Error: Invalid Response from COMServer.

Environment

  • /.

Solution

  • Happening when connecting to all devices:
    • The Console or End User Viewer are outdated or damaged:
      • (If a Technician) Run the Console installer obtained from Take Control: Downloads
      • (If an End User) Run the Viewer addon installer obtained from the End User Area
    • The MSXML parser is unstable, corrupted or missing on the local device (Windows Only):
      • Install Microsoft Core XML Services (MSXML) 6.0 Service Pack 1
  • Only happening when connecting to a specific device:
    • Change the computer name or the computer Group names to remove special character(s)
    • Attempt an update/repair of the Agent on the remote device, using the Agent Updater available from Take Control: Downloads
