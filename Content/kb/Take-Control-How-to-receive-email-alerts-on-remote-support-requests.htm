Take Control: How to receive email alerts on remote support requests?

Last Modified

Mon Apr 08 16:58 GMT 2019

Description

  • How to receive email alerts on remote support requests?
  • How to be notified on Missed Sessions with message?
  • How to be notified on answered Surveys?
  • Is there a way to be alerted on remote support requests without being logged into the Take Control console?

Environment

  • /

Solution

  1. Login on the Admin Area;
  2. Navigate to Profile > Integration;
  3. Select the NOTIFICATIONS tab;
  4. Select the option Enabled. Send to the following address:;
  5. Insert the desired email;
  6. Click the Save button.
  • Only one email is supported. Attempts to add several emails separated with a semicolon will not work.
