How to change applet file name on

Last Modified

Wed Dec 20 14:59 GMT 2017

Description

  • How to change applet file name on

Environment

Solution

  1. Navigate to your Admin Area
  2. Go to PROFILE > Applet Setup
  3. Select the tab 1. Name/logo
  4. Change the applet name under the "I. Applet Filename" area
  5. Click Save All
