Take Control: Agent not appearing in Console despite being installed

Last Modified

Fri Aug 09 17:48 GMT 2019

Description

  • Agent installed on remote device, states Service Ready, but I'm not able to view it in the My Computers list of the Console.

Environment

Solution

  • Refresh the list of devices in your console as follows:
    1. Login to your Technicians Console Application;
    2. Select the My Computers tab;
    3. Right click anywhere on the list and select Refresh from the context-menu (or use the F5 hotkey);
    4. The device should now appear if it is on your device list.
      • Please consider reviewing if it is on the wrong group.
  • Review if the Agent installed features your company logo. If it doesn't, a pre-existing Agent on the device, associated with another company, may have prevented the install. For more information see, [%=General.ProductName_tc-sa%]: How to Install Multiple Instances/Agents on the same Computer.
