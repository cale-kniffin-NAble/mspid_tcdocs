RMM Take Control Error: A connection to Take Control using the Engine gives the response - This server may be down or unavailable! Please try again later!

Last Modified

Mon Nov 11 13:45 GMT 2019

Description

  • When connecting to a device with Take Control using the engine, you see the following error message:
    • The server may be down or unavailable! Please try again later!

Environment

Solution

  • Socket error 10013 can be caused by antivirus or firewall software
  • Essentialy, a required outbound connection is being blocked
  • Socket error 10013 is a message which alerts that a port is blocked and/or unreachable
  • Ensure that the device has access to echo.global.mspa.n-able.com - this could be blocked by a firewall/router/av etc.

Article Number: 124408
