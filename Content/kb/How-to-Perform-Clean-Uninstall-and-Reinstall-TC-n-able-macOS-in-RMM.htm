How to Perform a Clean Uninstall and Reinstall of macOS in

Last Modified

Thu Dec 19 17:36 GMT 2019

Description

  • How to Perform a Clean Uninstall and Reinstall of macOS in .

Environment

Solution

  1. Turn off for the device in the :
    1. Right-click the device and select Edit Server / Edit Workstation.
    2. Go to Remote Access.
    3. Turn Off the Setting.
    4. Click OK to save.
      • Verify it is completely turned off by viewing the summary tab - - Not Installed.
  2. Locate and Delete the following file's and folders:
    1. /Applications/MSP Anywhere Agent TakeControl.app
    2. /Library\Logs\MSP Anywhere Agent TakeControl
    3. /Library\Logs\MSP Anywhere Installer
    4. /Users\<User Name>\Library\MSP Anywhere TakeControl
    5. /Users\<User Name>\Library\MSP Anywhere Installer
    6. /Library/MSP Anywhere Agent TakeControl
    7. /Library/LaunchDaemons/MSPAnywhereHelperTakeControl.plist
    8. /Library/LaunchAgents/MSPAnywhereServiceConfiguratorTakeControl.plist
    9. /Library/LaunchAgents/MSPAnywhereAgentPLTakeControl.plist
    10. /Library/LaunchAgents/MSPAnywhereAgentTakeControl.plist
    11. /Library/LaunchDaemons/MSPAnywhereDaemonTakeControl.plist
    12. /Library/PrivilegedHelperTools/MSP Anywhere Agent TakeControl.app
  3. Turn on for the device in the :
    1. Right-click the device and select Edit Server / Edit Workstation.
    2. Go to Remote Access.
    3. Turn ON the Setting.
    4. Click OK to save.
