: Launching Sessions without the Tech Console installed

Last Modified

Tue 12 Jan 17:57 GMT 2021

Description

  • Is it possible to launch sessions from the Admin Area without the Tech Console installed?

Environment

  • /

Solution

  • The Tech Console is always required to be installed, in order to perform any remote sessions - be it from the Tech Console or the Admin Area with the Remote Control Mode Active.
