Take Control: Console showing wrong language

Last Modified

Wed Feb 13 16:20 GMT 2019

Description

  • How do I change the language settings for the tech console?
  • The console's text is in another language, how can we change it?

Environment

  • /:
    • Tech Console 7.00.00 (Windows);
    • Tech Console 6.00.00 (macOS).

Solution

  • This setting can only be changed on the login screen of the console application:
    1. Logout of the Console (if already logged in) using the Log Off icon near the top right corner of the window;
    2. On the login screen, set the Language field to the intended language;
    3. You should now be able to login to the Console once again with the selected language.
