Take Control (All Versions): Send the CTRL+ALT+DEL Command on Agent Sessions

Last Modified

Tue Sep 10 11:26 GMT 2019

Description

  • How can we send the CTRL+ALT+DEL command to a remote device through a Standalone Take Control Agent Session, Session or Take Control session?
  • When I type CTRL+ALT+DEL locally, it triggers locally but not remotely, how can I trigger it remotely?

Environment

  • /
    • Take Control

Solution

  1. Start a Remote Session;
  2. Click on Remote Desktop on the top left of the Viewer
  3. Click on Send CTRL+ALT+DEL (or use the Shift+Ctrl+Del keyboard shortcut)
    • Please note that, for standalone Take Control, this option is ONLY AVAILABLE in Agent/Unattended Support connections to devices present in your My Computers list. This option is unavailable in Applet (attended support) sessions!
