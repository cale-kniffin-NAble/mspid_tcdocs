Is it possible to remove the label of the Operating System that Tech is using?

Last Modified

Tue Nov 13 14:15 GMT 2018

Description

  • In the Take Control Console, is it possible to remove the label of the Operating System that Tech is using?

Environment

  • /.

Solution

  • There is no option available to customize what is displayed in the Active Users menu of the Technician's Console - including the Operating System presented in front of the Tech name.
