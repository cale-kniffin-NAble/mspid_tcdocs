(All Versions): Trusteer Rapport blocking screen capture

Last Modified

Thu Oct 15 16:50 GMT 2020

Description

  • Trusteer Rapport blocking screen capture
  • A black screen with just a cursor appears

Environment

  • /
  • Take Control

Solution

  • A reboot seems to resolve issue
  • If end user is present have them close webpage causing issue
  • This issue is caused by browser security addons (such as Trusteer and it's Rapport browser addon) that forcefully stop image capture as a security feature
  • Please proceed as follows in order to resolve the issue:
    1. Run Browser in Safe Mode on remote device (not allowing any Addons)
    2. Individually enable addons to verify which one is forcefully blocking image capture
    3. Uninstall offending addon
      • Please note that you may need to request End User to perform uninstall, as many addons will block image capture during uninstall process as well
    4. Restart remote device
