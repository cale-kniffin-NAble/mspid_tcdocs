Check existing license and expiration date information

Last Modified

Mon Nov 19 07:20 GMT 2018

Description

  • This article provides information on how to check the existing license count for your company and license expiration date.

Environment

Solution

  1. Log in to the Admin Area.
  2. At the top of every page it will display the number of licenses (Licenses: nn) and their Expiration Date (Expiration: Month dd, yyyy).
