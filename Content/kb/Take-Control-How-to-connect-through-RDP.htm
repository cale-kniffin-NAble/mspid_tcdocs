Take Control: How to connect through RDP

Last Modified

Thu Jan 31 17:40 GMT 2019

Description

  • Steps on how to launch an RDP session through Take Control

Environment

Solution

  • Enable RDP Tunneling from the Take Control Agent
    1. Access the remote device;
    2. Open Take Control Agent configuration by double-clicking it in the Notification Area / system tray or searching for the application;
      • unlock Administration Mode by clicking the padlock icon on the top right corner;
    3. Click SETTINGS...;
    4. Check the option Enable RDP Tunneling;
    5. Click APPLY.
    • Please note this will only take effect on the next remote session. You will have to disconnect the current remote session and connect again before you can use RDP.
  • If after this you're still unable to RDP, please ensure that RDP is allowed on both devices (local device and remote device);
    • This Microsoft support article might be helpful.
