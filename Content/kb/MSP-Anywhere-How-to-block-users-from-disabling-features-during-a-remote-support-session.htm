: How to block users from disabling features during a remote support session?

Last Modified

Wed May 30 15:02 GMT 2018

Description

  • How to block users from disabling features during a remote support session?

Environment

Solution

  1. Login on the Admin Area;
  2. Go to MANAGEMENT > Technicians;
  3. Select the intended technician;
  4. Choose the Permissions tab;
  5. Under the Console Features portion, click Set Details;
  6. Uncheck the option Users can disable features;
  7. Click APPLY.

Article Number: 115044

Internal notes: Note this setting has to be done per technician, meaning it's an individual configuration not a general one.
