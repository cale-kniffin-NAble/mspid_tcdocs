How to download and install the Take Control Console

Last Modified

Thu Jan 30 11:01 GMT 2020

Description

  • How to download and install the Take Control Console;

Environment

  • /

Solution

  • Via Admin Area:
  1. Login to the Admin Area;
  2. Go to Downloads;
  3. On the Technician Console section, select the dropdown arrow to choose the desired OS;
  4. Click the OS button to download the installer;
  5. Run the installer executable.
  • Via Downloads page:
  1. Go to Take Control: Downloads.
  2. Click the icon for the Take Control Console version according to your OS;
  3. Run the installer executable.
