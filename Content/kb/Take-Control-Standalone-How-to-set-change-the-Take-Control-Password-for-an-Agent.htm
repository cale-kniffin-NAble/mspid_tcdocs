: How to set/change the Take Control Password for an Agent

Last Modified

Tue 12 Jan 09:45 GMT 2021

Description

  • How to set/change the Take Control Password for an Agent on a remote device.

Environment

  • /
    • Take Control Agent
    • Windows
    • macOS

Solution

  • Windows OS:
  1. Access the remote device;
  2. Restore/Load the Take Control Agent Configurator window;
    • It should be located in the systray, but you can also use the search option to locate it by name "Take Control Agent Configuration";
  3. Press the Advanced Setup button (cogwheel icon);
    • Should it be grayed out, press the Enter Administration Mode button on the top right corner to enable Admin Mode;
      • You'll need Administrative Privileges to proceed so a pop up window will appear informing about this;
  4. Select Security;
  5. Select the option Take Control Password;
  6. Type a password/new password under the Password field;
  7. Re-type the password above on the Confirm Password field;
  8. Click Apply.
    • To do this from the Admin Area without having to access the device directly please check the article [%=General.ProductName_tc-sa%]: How to change Agent settings
  • macOS
  1. Access the remote device;
  2. Restore/Load the Take Control Agent Configuration window;
    • On the top right corner (macOS Menu Bar), right click the Take Control Agent status icon and select Restore;
  3. Click the SECURITY button;
    • If the padlock symbol on the top right corner of the window is locked, click on it to Enter Administration Mode (or else the SECURITY button won't be visible);
  4. Select Take Control Password from the drop-down;
  5. Type the password you intend to set;
  6. Hit APPLY;
  • Next time you access the device, you'll be prompted to insert this same password.
