Remote Session can't be stopped

Last Modified

Tue Feb 04 09:30 GMT 2020

Description

  • Remote session didn't end when disconnected from viewer side
  • Remote device still shows blue box advising a session is in progress

Environment

Solution

  • Check that the session isn't still active on another device
  • If session is still running on another device, go to device and end session
