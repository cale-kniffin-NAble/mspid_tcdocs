How to enable 4k display compatibility view of the Take Control interface in Windows?

Last Modified

Wed Apr 29 12:29 GMT 2020

Description

  • Take Control's interface shows really small on computers with HD, 4K or UHD displays.
  • Screen capture appears distorted with in Take Control sessions.

Environment

Solution

  1. Right click on the Console application shortcut on the desktop.
  2. Click on Properties.
  3. Go to the Compatibility tab.
  4. Click the Change high DPI settings button.
  5. On the Choose the high DPI settings this program window, enable the Override high DPI scaling behavior.
  6. Select the System (Enhanced) option on the drop down list.
  7. Click OK, to close the high DPI settings window.
  8. Click Apply to save the changes.
  9. Close the Properties window by clicking OK
  10. Re-open the Console application from your desktop
Note: If in case there's no Desktop shortcut, the application is found in: C:\Users\jeff.malsi\AppData\Local\Take Control Viewer
