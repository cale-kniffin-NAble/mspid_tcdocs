: Linux Agent

Last Modified

Mon Jul 13 10:19 GMT 2020

Description

  • Does Take Control/ support a Linux Agent install?
  • How to generate a .deb or .rpm Agent installer?

Environment

  • /;
    • Console version 6.80.23 or above.

Solution

  1. Login to your Console;
  2. Select the My Computers tab;
  3. Click on the ADD COMPUTER button;
  4. Fill out the form as intended;
  5. Set Device Type to either:
    • Linux .deb (Debian, Ubuntu);
    • Linux .rpm (OpenSUSE, CentOS, Red Hat);
  6. Press GENERATE LINK - this will generate the download link and PIN Code for the .deb or .rpm file;
  7. On the remote Linux OS, download the agent using the generated link;
  8. Open the terminal and navigate the folder where the agent installer is located;
  9. Unzip the installer;
  10. Run the install.sh file with the command: sudo ./install.sh
    • The PIN code will be requested during the agent installation;
  • Please note that these Agents are currently still in development and may not yet have feature parity with our Windows and macOS/OSX Agents (but are intended to in future iterations).
