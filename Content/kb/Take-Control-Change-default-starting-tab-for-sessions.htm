Take Control: Change default starting tab for sessions

Last Modified

Mon Mar 11 17:34 GMT 2019

Description

  • How to change what tab opens up first when you connect to a remote session?

Environment

  • /:
    • Console v.7.00.00 onward.

Solution

  1. Start a remote session to a device;
  2. Click on the Preferences menu near the top right corner (vertical elipses);
  3. Hover your cursor over the Always start sessions in menu;
  4. Select the intended new default tab.
    • Note that you can also use the hotkey combination of Shift + Ctrl + F8 through F12 to achieve this resolve, witch each function key tied to it's respective tab.
    • This setting will persist for new sessions launched from this Console.
