Take Control: End User Limitations

Last Modified

Wed Sep 30 15:37 GMT 2020

Description

  • What can End Users make use of during a Take Control remote session from the end user area?
  • Can an End User use the System Shell?
  • Can an End User blank out the remote screen?

Environment

  • .

Solution

  • Although the remote access technology is the same, End Users will have significantly less tools to their disposal then technicians when performing their remote accesses from the End User Area.
  • An End User has access to:
    • Remote Desktop visibility and control;
    • File Transfer (which can be disabled from the Admin Area);
    • Restart & Shutdown options;
    • Start Audio or Video Call;
    • Disconnect/Pause Session;
    • Viewer visibility options such as Size/Zoom, Color and Change remote resolution);
    • Windows Commands (including "Send CAD");
    • Clipboard Control/Sharing;
    • Keyboard Mapping;
    • Remote Printing;
    • Locking Remote Operating System;
    • Connection Details;
    • Passportal interation button and menu (including autofill and copy username/password buttons).
