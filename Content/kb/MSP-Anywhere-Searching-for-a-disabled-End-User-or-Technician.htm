: Searching for a disabled End User or Technician

Last Modified

Wed Jul 25 13:07 GMT 2018

Description

  • How to find a disabled technician in the Management > Technicians list?
  • How to find a disabled End User in the Management > Technicians list?

Environment

Solution

  1. Login to your Admin Area;
  2. Navigate to Management > Technicians;
  3. To the right of the search bar, click the Filter icon and check the Disabled option;
  4. Use the Search Bar to the left to search for the Technician or End User's name;
    • You can use partial searching, such as searching for the common vowels in names (such as A - revealing all Annas, Andrews, Barbaras, etc.).
  5. The list should display all relevant Technicians and End Users, disabled or active. With disabled Technician and End User profiles appearing with some transparency.
