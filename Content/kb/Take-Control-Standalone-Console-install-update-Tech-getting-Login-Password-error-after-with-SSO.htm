: Console install/update/Technician getting Login/Password failed error after associating with SSO

Last Modified

Thu Apr 25 10:58 GMT 2019

Description

  • How to update the Take Control Tech Console .
  • Take Control Tech Console doesn't automatically update.
  • Where to obtain latest Take Control Tech Console.
  • Getting the following login error: "Login/password failed. Make sure you are entering the correct login and password and try again."

Environment

  • /

Solution

  • via Downloads page:
  1. Go to Take Control: Downloads.
  2. Depending on your operating system:
    • (Windows) Click Windows Console to download the latest version.
    • (macOS) Click Mac Console to download the latest version.
  3. Install the downloaded console.
  • via Admin Area:
  1. Login to the Admin Area;
  2. Navigate to the Downloads section;
  3. Under the Console section, click the button corresponding to the intended OS.
