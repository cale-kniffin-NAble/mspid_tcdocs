Selecting "Add Device" from Devices section in Admin Area only generates a link that installs the Agent onto one device

Last Modified

Wed Sep 11 13:15 GMT 2019

Description

  • Selecting Add Device from Devices section in Admin Area only generates a link that installs the Agent onto one device

Environment

    • Admin Area
    • Devices

Solution

  • The download link generated in Devices is custom created for the device for which it is generated and is not intended to be used to install the Take Control Agent onto any additional devices.
  • To download one agent that can be installed onto multiple devices:
    1. Select Downloads
    2. Under the Agent heading, click on the operating system for the devices onto which the Agent is going to be installed
    3. Select Save to download
    4. The downloaded executable can be run on multiple devices to install the agent onto them.
