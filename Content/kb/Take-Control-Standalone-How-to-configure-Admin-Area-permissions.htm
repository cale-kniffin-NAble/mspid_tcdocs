: How to configure Admin Area permissions

Last Modified

Tue May 21 08:50 GMT 2019

Description

  • How to configure Take Control Admin Area permissions.

Environment

  • /

Solution

  1. Login to your Admin Area;
  2. Navigate to Management>Technicians;
  3. Select the intended technician;
  4. Select the PERMISSIONS tab;
  5. Choose the ADMIN AREA sub tab;
  6. Set the options as intended for this technician;
    • You may also select a preset profile from the top left corner;
  7. Click the Save button.
