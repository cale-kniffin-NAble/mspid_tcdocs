: How to Rename an Agent from the Admin Area

Last Modified

Fri Jun 08 17:29 GMT 2018

Description

  • How to rename an agent from your Admin Area.
  • How to edit Agents details from your Admin Area.

Environment

Solution

  1. Navigate to your Admin Area
  2. Go on MANAGEMENT>My Computers>1. Computer List
  3. Select the desired device
  4. Under General edit the information at hand
  5. SAVE
