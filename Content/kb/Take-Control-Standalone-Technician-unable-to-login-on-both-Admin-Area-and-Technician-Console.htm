: Technician unable to login on both Admin Area and Tech Console

Last Modified

Wed 13 Jan 12:04 GMT 2021

Description

  • Getting the following error when logging in the Admin Area: "Technician invalid or inactive.Please check your username and try again. For more information, please contact the Take Control account administrator."
  • Getting the following error when logging in the Take Control Console: "Login/Password failed. Make sure you're entering the correct login and password and try again."

Environment

  • /

Solution

  • These symptoms are consistent with the tech profile having reached an expiration date, previously set for it;
  • Contact the Take Control Main Admin, or one of the Admins, to check the technician's profile settings.
