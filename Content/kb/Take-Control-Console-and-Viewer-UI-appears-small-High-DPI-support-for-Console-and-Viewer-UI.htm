: Console and Viewer UI appears small/High DPI support for Console and Viewer UI

Last Modified

Wed 13 Jan 12:42 GMT 2021

Description

  • On computers with very high resolution displays (QHD or 4K/UHD) Take Control's interface shows really small on the display;
  • Screen capture appears distorted with in Take Control sessions.

Environment

  • /

Solution

  1. Open the Console application;
  2. Close any Viewer windows (remote session windows);
  3. Select the Options menu on the top right corner (second option from the left, icon is a cogwheel);
  4. Enable Enable DPI Scaling at the Viewer;
  5. Close the Tech Console application entirely;
  6. Re-open the Tech Console application.
