Take Control: How to enable/disable View Only mode on a remote support session

Last Modified

Wed 13 Jan 12:21 GMT 2021

Description

  • Need to temporarily disable control of the remote device to make sure I don't interfere with my End User's inputs, how can I?

Environment

  • /.

Solution

  • While on a remote support session:
    1. Click on the Remote Desktop tab.
    2. Open the drop-down menu to the right of Remote Desktop (looks like a downward facing arrow);
    3. Hover over the View menu;
    4. Select View Only.
      • Alternatively, use the keyboard shortcut SHIFT + CTRL + L.
