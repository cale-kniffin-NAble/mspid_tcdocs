Description

  • With Dexpot installed, the is taking 30-45 seconds to get connected to a remote machine and during that time the technician's computer freezes/responds very slowly. Once the session connects, all seems to be fine.

Environment

  • /.
  • Supported Windows versions.

Solution

  • Verify if Dexpot is installed on the Tech's machine. If it is, please uninstall it.
    • Dexpot ia a Virtual Desktop application for Windows and will impact the Viewer performance's at the start of new sessions.
    • We currently do not have a workaround/bypass to this issue as it is caused by interference of a third-party application.
