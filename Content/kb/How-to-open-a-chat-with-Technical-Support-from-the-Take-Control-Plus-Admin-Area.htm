How to open a chat with Technical Support from the Admin Area

Last Modified

Thu Feb 28 22:03 GMT 2019

Description

  • How can i open a support chat in the Admin Area of ?

Environment

  • Admin Area

Solution

  • Login to Admin Area
  • On top right section of Admin Area, select circular icon with question mark in it.
  • Select Live Chat with Support
