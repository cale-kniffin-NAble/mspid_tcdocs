Take Control: Integrating the Applet on a website (Get Support Button)

Last Modified

Tue 12 Jan 10:32 GMT 2021

Description

  • How to integrate a download option for the Applet on your website for use with Take Control/?

Environment

  • / (Standalone);
    • Windows.

Solution

  1. Login to your Admin Area;
  2. Navigate to Profile > Integration;
  3. Select the YOUR WEB SITE tab;
  4. Copy the code template from the textbox;
    • Note that this template can be customized (such as adding your own custom image).
  5. Paste the code on your desired webpage's source code and publish it;
  6. Your website should now feature the "GET SUPPORT" button;
  • Your clients should now be able to perform support requests by clicking the GET SUPPORT button (and running the applet executable it downloads).
