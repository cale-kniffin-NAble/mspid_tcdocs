Take Control Require local user authorization on session start Windows Registry Values

Last Modified

Wed Nov 14 17:42 GMT 2018

Description

  • How can I change the Take Control Agent Values on the Windows Registry of the option Require local user authorization on session start?

Environment

  • Take Control Agent (Standalone)
  • Windows

Solution

  • If you wish to activate the option Require local user authorization on session start, you should navigate to the following registry path and set the value to 1
    • HKEY_LOCAL_MACHINE\SOFTWARE\WOW64NODE\Multiplicar Negocios\BACE\Settings\RequireLocalUserConfirmationOnSessionStart 1
  • If you wish to deactivate the option Require local user authorization on session start, you should navigate to the following registry path and set the value to 0
    • HKEY_LOCAL_MACHINE\SOFTWARE\WOW64NODE\Multiplicar Negocios\BACE\Settings\RequireLocalUserConfirmationOnSessionStart 0
