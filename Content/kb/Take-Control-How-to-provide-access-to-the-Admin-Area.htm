Take Control: How to provide access to the Admin Area

Last Modified

Mon Feb 24 11:28 GMT 2020

Description

  • This article instructs on how to grant Admin Area access for Take Control technicians.

Environment

  • /

Solution

  • Either the Main Admin or an Administrator must follow the steps below:
  1. Access the Admin Area;
  2. Go to Management > Technicians;
  3. Select the intended Technician;
  4. Click the PERMISSIONS tab;
  5. Select the ADMIN AREA sub-tab;
  6. Slide the toggle to enable the option Technician has access to the Admin Area;
  7. Click Save.
