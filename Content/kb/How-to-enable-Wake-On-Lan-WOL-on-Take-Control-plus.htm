Description

  • Is there a Wake-On-LAN feature with ?
  • Steps on how to enable WOL on .

Environment

  • Windows OS

Solution

  • Wake-On-LAN is one feature exclusively available in .
  • This feature is only available on Windows Agents.
  • This is not available with integrated Take Control versions on and .
  • The requirements for this feature are:
    1. Ensure Wake On Lan is supported on both local and remote devices - some laptop computers do not support WOL, or only support it while on AC Power. Some power supplies for desktop computers may also be incompatible;
    2. The computer BIOS also needs to be correctly set up. Look for a WOL option and enable it. This option may have different names ("Resume from my PCI Device", for instance), depending on the motherboard and BIOS version. If different options exist with similar names, test each one to find the correct option;
    3. Make sure both Take Control console and Take Control remote agents are updated to their latest version.
  • For more information, see Wake-on-LAN.
