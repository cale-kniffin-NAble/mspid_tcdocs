Description

  • Steps on how to change the default path for session recordings on Take Control

Environment

  • /
    • Take Control

Solution

  • During a remote support session:
  1. Select the vertical ellipsis icon (top right corner of the )
  2. Select Video Storage Location > Custom
  3. Choose the desired folder
  4. Click OK
