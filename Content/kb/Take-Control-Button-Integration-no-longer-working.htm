Take Control: Button Integration no longer working

Last Modified

Fri Jul 05 13:41 GMT 2019

Description

  • Clicking the "Get Support" button integrated into my web page no longer downloads an Applet;
  • The "Get Support" button does nothing or returns an HTTP 503 error;
  • Get Support button URL directs to a "secure.corporate.beanywhere.com" URL.

Environment

  • /.

Solution

  • Hosts under the "*.beanywhere.com" domain for integration options, such as the Website integration of the "Get Support" button are being deprecated in favor of hosts under the *.swi-tc.com domains;
  • Please update the integration code in use in your web site's HTML code with the more recent integration available in your Admin Area under Profile > Integration > YOUR WEB SITE.
