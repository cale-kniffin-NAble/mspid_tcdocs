Take Control (All Versions): EntryProtect is blocking this screen capture

Last Modified

Fri May 31 11:31 GMT 2019

Description

  • When performing a remote session to a machine and on the you click on Remote Desktop, you get a white page with the message
    • Error: “EntryProtect is blocking this screen capture"

Environment

  • Take Control

Solution

  • Check if SafeNet is installed on the remote device and if so:
    • Install a version of SafeNet without the EntryProtect Feature (since this feature is designed to block screen capture to any remote sessions established to the machine)

    Article Number: 130133

    Internal notes: Simillar issues may occur with Trusteer, Webroot, McAfee, Kaspersky
