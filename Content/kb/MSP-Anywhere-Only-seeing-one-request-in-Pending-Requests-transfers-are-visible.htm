: Only seeing one request in Pending Requests, transfers are visible.

Last Modified

Mon Jul 23 16:52 GMT 2018

Description

  • I can only see the oldest pending request in the Pending Requests tab of the Console;
  • Transferred requests show up fine, other pending requests do not.

Environment

Solution

  • The behavior described matches the intended behavior for the "Show only one request at a time" setting that can be enabled per technician, please review/disable this setting as follows:
    1. Login to your Admin Area;
    2. Navigate to Management > Technicians;
    3. Select the relevant Technician's profile;
    4. Select the Permissions tab;
    5. Select the Console Features submenu;
    6. Review/Disable the Show only one request at a time option;
    7. Save (bottom right corner) when done;
    8. (If already logged into the Console, please logout and then) Login to your Console to pick up the new settings.
