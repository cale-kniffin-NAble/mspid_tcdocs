Take Control: Getting "Invalid Credentials" error when trying to login on the end user area

Last Modified

Tue Mar 26 12:23 GMT 2019

Description

  • Unable to login on the end user area.
  • Getting "Invalid Credentials" error.
  • How to reset end user password.

Environment

Solution

  • The "Invalid Credentials" message only occurs if the wrong credentials are at place.
  • In this scenario i'ts advised to reset the end user's password, as follows:
  1. Navigate to the End User area;
  2. Click the Forgot Password? link;
  3. Input your username when prompted and press Send;
  4. You'll be directed to the relevant Password Reset page, confirm the username there and press RESET YOUR PASSWORD;
  5. You'll receive an email with instructions to reset your password.
