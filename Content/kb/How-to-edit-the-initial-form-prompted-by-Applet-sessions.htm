How to edit the initial form prompted by Applet sessions

Last Modified

Thu Jan 31 13:44 GMT 2019

Description

  • How to edit the initial form prompted by Applet sessions?

Environment

  • /.

Solution

  1. Login to your Admin Area;
  2. Navigate to Profile > Applet Setup > INITIAL FORM;
  3. Edit the fields as intended;
  4. Press the Save button near the bottom right corner.
