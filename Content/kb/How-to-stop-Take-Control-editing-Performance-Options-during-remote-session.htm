How to stop Take Control editing the Performance Options of the remote device during remote session?

Last Modified

Fri Feb 14 02:51 GMT 2020

Description

  • Remote Device setting for Performance Options > Visual Effects is Custom.
  • When in Remote Session, some selection such as Smooth Edges of screen fonts and others get unticked.

Environment

Solution

  • This is default behavior when "Optimize Desktop Settings when in Session." is enabled. To update the configuration:
  1. As default to future installation of agents:
    1. Login to your Admin Area
    2. Navigate to Profile > Agent Setup
    3. Select the Settings tab
    4. Untick "Optimize Desktop Settings when in Session."
    5. Save when done (bottom right corner);
      • These settings will only apply to NEW installations of the Agent performed after this point. Follow steps below to update existing agents.
  2. Per Take Control Agent:
    1. Connect to the remote device
    2. Restore/Run the Take Control Agent Configuration window;
      • It should be in the Windows system tray, but you can also use the search it by name "Take Control Agent Configuration"
    3. Click on the Padlock icon to the top right of the window to unlock the agent, you should be able to view the Settings button after this.
    4. Click on Settings.
    5. Untick "Optimize Desktop Settings when in Session."
    6. Press APPLY
