Take Control: Access System Info through the Admin Area

Last Modified

Mon Dec 28 11:10 GMT 2020

Description

  • How to check System Info outside a Take Control remote session?

Environment

  • /

Solution

  1. Login to your Admin Area ;
  2. Navigate to Devices;
  3. Select the intended Agent/Computer from the device list;
  4. Select the SYSTEM tab (below the list).
    • Note that only devices with "Enable monitoring" active will have this tab available.
