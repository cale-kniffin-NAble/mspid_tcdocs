: How to incorporate the PIN code for applet sessions on a website

Last Modified

Thu May 14 12:35 GMT 2020

Description

  • Steps on how to incorporate the PIN code for applet sessions on a website.
  • How to use the Landing Page API.

Environment

  • /

Solution

  1. Login to the Admin Area;
  2. Navigate to Profile > APIs;
  3. Select the START APIS tab;
  4. Under the "Landing Page API" area choose the desired idiom in the Language: drop down;
  5. Choose PIN Code on the Code Type: drop down;
  6. Click the button Generate;
  7. Incorporate the generated HTML code on the website or customize it as desired.
