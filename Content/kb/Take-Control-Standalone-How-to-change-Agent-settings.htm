: How to change Agent settings

Last Modified

Tue Nov 03 10:17 GMT 2020

Description

  • How to change settings on an existing Agent install?
  • How to bulk change agent settings?
  • How to change agent setting within a group?

Environment

  • /

Solution

  • Note that agent settings can be changed in bulk from the Admin Area only for Take Control Windows agent v.7.00.12 and onwards:
  • IMPORTANT: For Take Control macOS agents it's not possible to bulk change the settings in a centralized way. This needs to be done per agent and directly on the agent's configurations.
    • By directly accessing the remote device:
      • Windows OS
      1. Access the remote device;
      2. Restore/Load the Take Control Agent Configuration window;
        • It should be located in the systray, but you can also use the search option to locate it by name "Take Control Agent Configuration";
      3. Press the Advanced Setup button (cogwheel icon);
        • Should it be grayed out, press the Enter Administration Mode button on the top right corner to enable Admin Mode;
          • You'll need Administrative Privileges to proceed so a pop up window will appear informing about this;
      4. Select Security/Settings;
      5. Apply the intended changes;
      • macOS
      1. Access the remote device;
      2. Restore/Load the Take Control Agent Configuration window;
        • On the top right corner (macOS Menu Bar), right click the Take Control Agent status icon and select Restore;
      3. Click the SECURITY/SETTINGS button;
        • If the padlock symbol on the top right corner of the window is locked, click on it to Enter Administration Mode (or else the SECURITY and SETTINGS buttons won't be visible);
      4. Click APPLY to ensure the changes are assimilated;
    • From the Admin Area:
      • Note this option is only available for Windows OS and Take Control Agents from version 7.00.12 onwards;
      1. Login to the Admin Area;
      2. Navigate to the Devices menu;
      3. Select the intended device(s) from your device list;
      4. Click More > Apply Configuration;
      5. On the Target Devices section select Apply to selected or Apply to groups as intended;
      6. Choose the desired actions on each setting according to the standard symbol meanings;
        • - = Do not change (Leave as is)
        • <blank> = Uncheck (Disable)
        • Check mark = Checked (Enable)
      7. Click Apply to deploy changes.
      • Usually it takes up to 1 or 2 minutes to propagate these changes but this can vary according to device specs as well as the local and remote network environments at place.
