Not receiving texts / emails for 2FA

Last Modified

Wed Apr 29 19:34 GMT 2020

Description

  • I'm not receiving two-factor authentication text message / email when trying to login

Environment

Solution

  • does not send text messages or email messages for 2FA (two-factor authentication).
  • 2FA codes are generated in the 2FA App used by the user to setup 2FA for the username in question.
  • Check the 2FA app used to setup 2FA for the user for the 2FA code required to login
