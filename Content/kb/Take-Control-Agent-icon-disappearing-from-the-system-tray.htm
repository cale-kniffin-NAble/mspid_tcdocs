Take Control: Agent icon disappearing from the system tray

Last Modified

Wed Nov 07 13:53 GMT 2018

Description

  • Take Control agent installed and showing on the Console's My Computers list as online but the system tray icon is missing on the remote end.

Environment

  • .

Solution

  • This situation might happen due to the Take Control Agent Configuration process not being launched properly.
  • To fix this, please follow the steps below:
  1. Access the remote device;
  2. Navigate to C:\Program Files (x86)\BeAnywhere Support Express\GetSupportService;
  3. Run the BASupSrvcCnfg.exe file;
  4. The Agent Configuration process will be launched and the system tray icon should show as expected.
  • Alternatively you can run the agent updater over the already installed agent as follows:
    1. Access the remote device;
    2. Go to Take Control: Downloads.
    3. Click Agent Updater to download.
    4. Run the Agent updater executable;
    5. The Take Control agent will be launched as well as all the related processes, including the Agent Configuration process, causing the system tray icon to appear as expected.

Article Number: 125395

Internal notes: If for some reason it's not possible to perform the steps above, a new agent installer will need to be generated as per article 000049340
