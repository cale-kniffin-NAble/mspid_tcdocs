Check for running sessions in the admin area

Last Modified

Wed Nov 30 21:44 GMT 2016

Description

This article details where to check the real count of all the running sessions in .

Environment

Solution

  1. Log in to the Admin Area.
  2. Go to Reports > Running Sessions.
  3. It would display all the technicians currently online.
  4. From here, you can force log-out the technicians to free up concurrent license.
