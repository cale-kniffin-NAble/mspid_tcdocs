Take Control: Can't accept incoming requests from Admin Area without a license

Last Modified

Fri Apr 05 14:27 GMT 2019

Description

  • When selecting an Incoming Request to connect to from the Admin Area, we receive the following error:
    • "Your license does not support this feature".

Environment

  • .

Solution

  • The Remote Control mode's ability to accept incoming sessions from the Admin Area is only available to licenses.
