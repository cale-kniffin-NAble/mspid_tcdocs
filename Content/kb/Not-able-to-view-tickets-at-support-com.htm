Not able to view tickets at success.n-able.com

Last Modified

Mon Dec 18 20:56 GMT 2017

Description

  • Partner is able to log into support but is not able to view tickets

Environment

  • success.n-able.com
  • Customer Portal

Solution

  • Make sure the web browser and javascript have been updated
  • Disable any add blocker or plugin that is being used (or use incognito mode)
  • Clear the browser cache
  • Use another browser
