Take Control: How to disable pop-up chat window on message?

Last Modified

Wed Nov 14 12:02 GMT 2018

Description

  • How to disable chat window while on a remote session?
  • How to disable pop-up chat window on message?

Environment

  • /

Solution

  • This setting must be done individually on each Take Control Agent, as follows:
  1. Access the remote device (either by a Take Control session or onsite);
  2. Open the Take Control Agent interface by double clicking the Take Control Agent icon located on Systray;
  3. The Take Control Agent interface window will pop;
  4. On the top right corner, click the blue colored icon saying pop-up chat window on message when hovering over it;
  5. After this, the chat window will not fully open the next time a message is sent as a small pop up notification will appear on the bottom right corner instead.
