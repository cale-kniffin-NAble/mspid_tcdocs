: Mobile Access Features

Last Modified

Mon Dec 28 17:36 GMT 2020

Description

  • What can I do when connected to an Android or iOS device?
  • Are connections to mobile devices identical to connections to Windows or macOS devices?

Environment

    • Take Control iOS Applet
    • Take Control Android Applet

Solution

  • During a remote session to a mobile (Android or iOS) device you'll be able to:
    • View the video from both cameras:
      • Select the device cameras as you would select a monitor (Monitor button on the bottom right corner)
    • Ask the user for a screenshot, using the Remote Control tab
      • Please note that live visibility of the Remote Desktop is only possible when connected to Android devices that support such a feature - if available, it'll appear once you press the Monitor button on the bottom right corner after viewing the device Cameras, if unavailable this will request a new screenshot from the End User
      • Remote Control and Live visibility of the Remote Desktop is currently unavailable on remote sessions to iOS devices
    • Send and receive files to the private storage area of the Applet using the FileTransfer tab
      • The user can also store files in the private storage area
    • Upload configuration profiles with ease, setting up e-mail, wireless configurations, etc.
    • Chat with users in real time.
    • The end user feature is not currently available on mobile devices.
