Description

  • How to limit Admin Area and/or Tech Console access to specific sets of IP's?
  • Error: Your current IP address is not authorized to access this resource. Please make sure you are connecting from an authorized IP address and try again.

Environment

  • /.

Solution

  • If you're receiving the following error" Your current IP address is not authorized to access this resource. Please make sure you are connecting from an authorized IP address and try again." - please connect from a device that reports the expected public IP address or have your Master Admin reach out to our Tech Support.
  • Note that you can always set the IP restriction as follows:
    1. Log in to the Admin Area webpage;
    2. On the Profile menu, select Identification;
    3. Now select the Security tab;
    4. Under Access Control, you've two distinct options: Limit the access to Admin Area by IP and Limit the access to Tech Console by IP;
    5. Chose the option(s) you wish and identify the addresses in the box(es) allocated, making sure that the IP's are separated by commas;
      • Please note that should ALWAYS input the Public IP of the address you're attempting to connect from, private IP addresses cannot be validated.
    6. Save the setting(s).
