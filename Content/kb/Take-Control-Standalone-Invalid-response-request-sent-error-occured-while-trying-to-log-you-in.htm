: An Invalid response to your request was sent/An error occured while trying to log you in

Last Modified

Fri Jan 11 11:57 GMT 2019

Description

  • Unable to login to Take Control Tech Console on a specific computer while able to login on any other computer in the same network;
  • Returning Error: "An Invalid response to your request was sent. Please try again in a few moments or contact us if the problem persists"
  • Returning Error: "An error occured while trying to log you in. Please try again. If the problem persists please contact us. Thank you!"

Environment

Solution

  • Download and install the following Microsoft Package:
    • https://www.microsoft.com/en-us/download/details.aspx?id=15697
  • If the issue still persists, uninstall the current console and perform a new console install by downloading the latest version of the Take Control Windows Console available at Take Control: Downloads.
