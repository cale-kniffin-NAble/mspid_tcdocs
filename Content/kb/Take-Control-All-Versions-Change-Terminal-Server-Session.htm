Take Control (All Versions): Change Terminal Server Session

Last Modified

Thu May 28 13:24 GMT 2020

Description

  • Need to switch between sessions when connected to a Terminal Server, can I do this? How?

Environment

  • :
  • :
    • Take Control;
    • .

Solution

  1. Launch a remote session to the intended Device;
  2. Click on the Remote Desktop tab;
  3. Open the drop-down menu to the right of Remote Desktop (looks like a downward facing arrow);
  4. Hover over the View menu;
  5. Hover over Change TS Session and click on the intended active user session from the list.
    • Note that this option will be disabled/grayed out when accessing a device via standalone Take Control's Attended Access Applet or for standalone Take Control (non-) accounts.
