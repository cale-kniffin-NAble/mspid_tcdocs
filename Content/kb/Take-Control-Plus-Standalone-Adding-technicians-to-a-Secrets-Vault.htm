: Adding technicians to a Secrets Vault

Last Modified

Wed Jul 01 14:41 GMT 2020

Description

  • How to add Technicians to a Secrets Vault?
  • How to share access to a Secrets Vault to other employees?

Environment

  • :
    • Secrets Vault.

Solution

  1. Login to your Admin Area as an Admin or the Main Admin;
  2. From the left-hand pane, select Management > SecretsVaults;
  3. Select the intended vault from the list on the left;
  4. Select the TECHNICIANS tab on the right, under the Secrets list for that vault;
  5. Select the Add button near the bottom right corner, under the list of Technicians;
    • Note that you may need to use the scrollbar on the right side of the Technician's list to scroll down in order for the Add button to be visible.
  6. Check the intended technicians you intended to grant access to the Vault;
    • Technicians marked with an asterisk (*) currently do not have access to the Admin Area or to the Secrets Vaults repository. Inviting them to a Vault will automatically grant them these permissions!
  7. Select Next;
  8. Provide the TEMPORARYSHAREDSECRET to the intended technicians, be sure to save a copy of it for recovery purposes as well;
  9. Check I have copied the Temporary Shared Secret and select Next;
  10. Input this Vault's VaultPassword and select Save;
    • If the password is incorrect, you'll receive an Error: Failed to Validate Technician.
  11. If the password was correct, you'll receive a prompt indicating how many Technicians were added successfully, select Close to finish this procedure.
    • Please note that the Technicians will simply have been Invited. They'll need to complete their invitation using the Temporary Shared Secret you've provided them.
