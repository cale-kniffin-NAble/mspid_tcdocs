Take Control: Support cases show up as UNKNOWN/Set Techs not be able to view support request details

Last Modified

Thu Jan 24 19:47 GMT 2019

Description

  • How to setup a Tech to not be able to view incoming case details (name, device, etc.)?
  • Incoming requests are showing up as UNKNOWN on my tech's console.

Environment

  • /.

Solution

  • You can enable or disable a setting for each technician that will manage whether or not they'll be able to see incoming request details (if not, they'll see them as UNKNOWN). This can be set as follows:
    1. Login to your Admin Area;
      • Be sure to login as an Admin with sufficient privileges to change Tech Permission. If your login lacks these permissions, request this procedure to one of your Admins or the Master Admin.
    2. Click on Management on the left hand pane;
    3. Click on Technicians;
    4. Select the intended technician's profile from the list;
    5. Select to the PERMISSIONS tab to the right of the list, once the profile is selected;
    6. Select the CONSOLE FEATURES sub-tab within the permissions menu;
    7. Set Show Request Details on to ensure the Tech is able to see incoming request details or off to ensure they can only view them as UNKNOWN;
    8. Save when done (bottom right corner).
      • In order to have these settings apply immediately after saving, have the tech log out and log back into the Tech Console - should they be logged in prior.
