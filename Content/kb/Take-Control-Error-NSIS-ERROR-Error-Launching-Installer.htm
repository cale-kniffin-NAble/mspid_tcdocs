Take Control: Error: NSIS ERROR "Error Launching Installer"

Last Modified

Tue Nov 20 12:43 GMT 2018

Description

  • When trying to install a Tech Console or an Agent an NSIS error occurs
    • NSIS ERROR: "Error Launching Installer"

Environment

Solution

  • When we try to run the installer of the Take Control Tech Console or the Take Control Agent, if we get the error: NSIS ERROR "Error Launching Installer", this could mean that:
    • The installer is corrupted, and we should download a new installer and retry the installation
    • The Windows User Account has some level of corruption, and we should Loggon with a different user account and retry the installation
    • The Windows System has some corrupted files, and we should run the command sfc /scannow in order to search for corrupted system files. If the system is unable to auto-heal a system reinstallation may be necessary
