Take Control: Script not running, results showing blank output

Last Modified

Wed Mar 13 14:34 GMT 2019

Description

  • Have a script that was sent out to a device successfully but didn't seem to do anything;
  • In the Admin Area, under Reports > Script Jobs, selecting the Script Job from the list and checking the Results tab shows no Script Exit code and only a blank text box.

Environment

  • /.

Solution

  • Review if the chosen Script is Enabled as an Disabled script will not run on the Task:
    1. Login to your Admin Area;
    2. Navigate to Management > Scripts;
    3. Select the intended script;
    4. Under the SCRIPT tab (on the south pane), set the Enabled setting from OFF (Gray) to ON (Green).
