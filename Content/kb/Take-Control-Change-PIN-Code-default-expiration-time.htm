Take Control: Change PIN Code default expiration time

Last Modified

Thu Feb 14 13:47 GMT 2019

Description

  • By default, PIN Codes are set to expired within 12 hours of being created - this falls within the design of Applets serving to start an Attended Support request from the client's end without installing a long-term program on their device - how can we extend or reduce that expiration time?

Environment

  • /.

Solution

  1. Login to your Admin Area;
  2. Navigate to Profile > Identification;
  3. Select the PIN/CALLING CARDS tab;
  4. Set the PIN Codes expire after (...) From its creation value to your preferred setting (from 1 hour to 7 days);
  5. Save when done.
