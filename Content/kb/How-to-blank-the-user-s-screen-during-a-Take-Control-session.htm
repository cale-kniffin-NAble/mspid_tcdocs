How to blank the user's screen during a Take Control session?

Last Modified

Wed Mar 04 22:32 GMT 2020

Description

  • How to use Blank Screen?
  • How to sleep the display of the remote user?

Environment

  • /
    • Take Control

Solution

  1. Start a Take Control session to the device in question;
  2. Click on the Remote Desktop tab (on the top left corner of the Viewer's window);
  3. Click on the Commands submenu;
  4. Click on Show Blank Screen (or use the Shift+Ctrl+N hotkey combination);
    • Please note that this may not be possible depending on the monitor in use (such as CRT monitors) and power settings (if the user is set to lock windows on display sleep, for example, attempting to sleep the monitor will lock the user account).
    • Please note that, within the standalone product, End Users will not have access to this feature.
