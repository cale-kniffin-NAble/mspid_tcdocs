Take Control: How to Export a Session History Report

Last Modified

Fri Aug 23 10:23 GMT 2019

Description

  • How to generate Session History reports on Take Control/?
  • How to filter the information on a Session History report?
  • How to export a report to excel?

Environment

  • /.

Solution

  1. Go to your Admin Area;
  2. Navigate to Reports>Session History;
  3. Select the desired time-span from the Date: drop down menu;
    • Please note that you'll only be able to filter for sessions that are up to 2 years old.
  4. Select the desired sessions to be exported;
  5. Click the Export button on the top right corner of the Admin Area page;
  6. Chose between Selected only or Entire resultselt;
  7. Click the desired export option by clicking the Export button or the Export with chat button.
    • Please note that you will only be able to export up to 5000 session records in one export. If you wish to export more than 5000 results, please do so in separate exports.
    • Please note that the export format (xls) is intended to be opened via a compatible desktop Excel application - as web based Excel will not be able to perform the required file conversion.
