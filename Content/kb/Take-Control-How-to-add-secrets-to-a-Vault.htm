Take Control: How to add secrets to a Vault

Last Modified

Mon Oct 22 15:39 GMT 2018

Description

  • Steps on how to add secrets to a vault

Environment

Solution

  1. Login to your Admin Area;
  2. Navigate to Management > Secrets Vaults;
  3. Select the desired Vault;
  4. Click the Add button on the Secrets area (top right corner);
  5. Fill the form as intended;
  6. Click Add another or Continue, as desired;
  7. Type Vault Password and click the Save button
