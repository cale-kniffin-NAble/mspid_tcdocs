Take Control: Generating an Agent Installer for a Specific Group

Last Modified

Mon Dec 30 15:02 GMT 2019

Description

  • When generating a Take Control Agent installer, how to ensure device is to be listed under an existing group after install?

Environment

  • /.

Solution

  1. Login to your Take Control Desktop Console;
  2. Select the My Computers tab;
  3. Select the Add Computer button;
  4. Fill out the form to the right side of the window;
    • Note that only the Computer Name field is mandatory, but please review all fields before proceeding!
  5. Select the "Root Group\" entry in front of the Group field;
  6. Select the intended Group or Sub-Group from the presented list and select OK
  7. Select Generate Link;
  8. You may now COPY or EMAIL the download link for the Agent installer.
    • For MacOS installs, you'll also need the generated Computer PIN Code above the Installation Link!
