: Create Computer Groups

Last Modified

Tue 12 Jan 16:43 GMT 2021

Description

  • How to create Computer Groups in the Take Control Tech Console?
  • How to create new Agent Groups in the Take Control Admin Area?

Environment

  • /

Solution

  • You can add new Agent groups through either:
    • The Technician's Console:
      1. Login to your Take Control Technician's Console application (as a technician with sufficient privileges);
      2. My Computers;
      3. Right-Click anywhere on the list;
      4. Select New Group or press Ctrl + N;
      5. Type the intended group name;
      6. Press OK.
    • The Admin Area:
      1. Login to your Admin Area (as a technician with sufficient privileges);
      2. Navigate to Devices (left pane);
      3. Press on Add to the right of Group (right side of the window);
      4. Type the intended group name and press Add.
