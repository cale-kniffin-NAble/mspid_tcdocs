: How to turn an end user into a technician

Last Modified

Tue May 21 08:56 GMT 2019

Description

  • Steps to be taken to turn an end user into a technician.

Environment

Solution

  1. Navigate to the Admin Area;
  2. Select Management > Technicians;
  3. Select the intended client (end user);
  4. Untick the option User is a customer
  5. Click the Send Invitation button under Single Sign On (SSO) text;
  6. Click Save;
  7. Then navigate to the technician's email inbox, where you'll received the SSO invitation;
  8. Open the invite email and click the Create SSO Account link;
  9. Fill the form with the tech's details (First name, Last name) and set a password;
  10. Click the button CREATE ACCOUNT and then CONTINUE.
