How to change the technicians email address in the Admin Area for Take Control Standalone?

Last Modified

Thu Nov 26 10:41 GMT 2020

Description

  • I've accidentally registered a Tech with the wrong e-mail address
  • How to change the technicians email address in the Admin Console for Take Control/?

Environment

  • /

Solution

  • NOTE the following procedure can only be done BEFORE a technician performs their invitation via Take Control
  • Once a Technician has already performed their first login via , it will no longer be possible to change their username (the field will be grayed out and not editable):
    1. Go to the Admin Area
    2. Navigate to Management > Technicians
    3. Select the desired technician
    4. Choose the TECHNICIAN tab
    5. Change the username as intended on the Username (Email): area
    6. Click the Save button

Article Number: 131670

Internal notes: PLEASE NOTE: USERNAME in the context of Take Control is the EMAILTHE NAME CAN BE CHANGED USING THE USER MENU ON THE TOP RIGHT CORNER AND CHOOSING "EDIT YOUR DETAILS"This is valid for all technicians except the main admin. Please check article ID: 000049592
