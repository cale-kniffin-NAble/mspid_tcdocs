: Force log-out a technician

Last Modified

Tue Sep 22 17:18 GMT 2020

Description

  • How to allow technicians to be disconnected by others on login (assuming the concurrent seat limit is reached)?
  • How to manually logout a technician?

Environment

  • .

Solution

  • This feature is ONLY available for licenses!
  • Using the manual force-logout option:
    1. Login to your Admin Area;
      • Must be done as an Administrator!
    2. Select the User Menu near the top right corner (where you can see your name and image);
    3. Select Licensing Information;
      • Alternatively, you may access the Licensing Information page via Management > Licenses.
      • NOTE: This option will only be available if your profile has sufficient privileges!
    4. Navigate to ACTIVE LICENSES;
    5. Press the Force Logout button to the right of the desired Active Technician;
      • NOTE: This option requires the Technician to have "Can be disconnected" enabled on their Permissions.
    6. The Technician should no longer be logged into their console.
  • Setting up Technicians to be able to disconnect other Technicians:
    1. Login to your Admin Area
    2. Navigate to Management>Technicians;
    3. Select the Technician's Profile and it's Permissions tab;
    4. Select the Global subtab;
    5. Enable the Can disconnect any other techs feature;
    6. Ensure that the other technicians have the Can be disconnected option enabled
    7. Hit Save.
    • NOTE: This option can not be enabled for the Main Admin;
    • Technicians cannot disconnect the Main Admin even if they're Administrators;
    • Only the Main Admin can disconnect themselves.

Consequentially, this needs to take into account the Technician permissions such as whether or not they can be disconnected.The old Admin Area does not feature a Remote Access mode and did not need to take these permissions into account (as Forced Logout would always just be used to force the technician's logout from the Console application or mobile app)
