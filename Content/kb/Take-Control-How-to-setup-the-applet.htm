: How to setup the applet

Last Modified

Wed 13 Jan 12:35 GMT 2021

Description

  • Steps on how to configure applet details

Environment

  • /

Solution

  1. Login to your Admin Area;
  2. Go to Profile > Applet Setup;
  3. Change the options as desired;
  4. Click Save when done;
  • Take into account this will only affect applets generated after these changes were made as previously generated applets will not be affected.
