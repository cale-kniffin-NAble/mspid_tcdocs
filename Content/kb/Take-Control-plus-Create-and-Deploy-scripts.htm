: Create and Deploy scripts

Last Modified

Fri Apr 03 13:33 GMT 2020

Description

  • How to create and deploy Visual Basic Script, Powershell or Batch scripts through standalone?

Environment

Solution

  • Creating the Script:
    1. Login to your Admin Area;
    2. Navigate to Management > Scripts;
    3. Click the +Add script button;
    4. Fill the script Name and Description;
    5. Select the Script Type on the drop-down (PowerShell, Visual Basic Script or Batch);
    6. Click Add;
    7. Select the newly created script from the list;
    8. On the chart below the script list, enable the script by switching the toggle in the SCRIPT tab;
    9. Save changes;
    10. Select the CONTENT tab;
    11. Select Explicit Script or External Script as desired;
      • Write the Script Body or use the Script URL and Script MD5 according to the previous choices;
    12. Save changes;
    13. Select the REQUIRED FILES tab and add the desired files by clicking the button Add new button;
  • Deploying the Script:
    1. Login to your Admin Area;
    2. Navigate to Devices;
    3. Select the device on which you want to deploy the script;
    4. Click More > Deploy Script;
    5. Fill the fields on the form corresponding to Job name, Script (to choose from the existing ones), Start on, Run as, Deployment and Notifications sections as desired;
    6. Click Deploy script;
    • The script will be queued for deployment and its status can be checked on the Admin Area in Reports > Script Jobs.
