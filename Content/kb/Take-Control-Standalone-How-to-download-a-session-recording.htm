: How to download a session recording?

Last Modified

Tue 12 Jan 17:25 GMT 2021

Description

  • How to download a recorded session from the Admin Area?

Environment

  • /.

Solution

  1. Login to your Admin Area;
    • If you're already logged into your Admin Area, click the above link and skip to step 4.
  2. From the left pane, click on Reports;
  3. Select Session History;
  4. Click on the relevant session from the list;
    • Note that you may need to use the Filters option, available near the top left corner of the session list pane, in order to filter for the intended result.
  5. On the right pane displaying the session summary, left-click the Videos(X) drop-down;
    • Note that X should instead display the number of recordings from that same session - if listed at 0, there were either no recordings performed or the session recordings have yet to upload!
    • Please note that recordings are only available for download up to 6 months after they were performed.
    • Alternatively, you can also select Show in Reports Page for a full page Session Summary, you will need to then select the Video(X) tab.
  6. Click on the Download Icon under the intended recording;
    • Note that the recording format will be FLV and will require a Media Player with the relevant Codec to play.
