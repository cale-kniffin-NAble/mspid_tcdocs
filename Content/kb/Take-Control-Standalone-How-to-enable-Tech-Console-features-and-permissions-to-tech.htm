: How to enable Tech Console features and grant permissions to technicians

Last Modified

Wed Jul 01 15:15 GMT 2020

Description

  • How to enable Tech Console features and grant permissions to technicians on Take Control Standalone?

Environment

  • /.

Solution

  1. Login to the Admin Area (as either the Main Admin or a Technician with sufficient privileges);
  2. Go to Management > Technicians;
  3. Select the technician you want to grant access;
  4. On the PERMISSIONS tab, select the CONSOLE FEATURES sub tab;
  5. Enable/Disable the intended options;
  6. Save when done (bottom right corner).
  • If the tech at hand is logged into the Tech Console during permissions change, please logout and log back in to experience the applied changes.
