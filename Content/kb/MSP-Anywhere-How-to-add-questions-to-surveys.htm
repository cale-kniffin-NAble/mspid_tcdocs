: How to add questions to surveys

Last Modified

Fri Mar 09 17:35 GMT 2018

Description

  • How to add questions to surveys on ?

Environment

Solution

  1. Go to your Admin Area;
  2. Navigate to MANAGEMENT>Surveys;
  3. Select the intended survey;
  4. Select the 2. List of questions tab;
  5. Click on the NEW QUESTION button;
  6. A window will appear with the following fields:
    • New Question - Write the desired question here;
    • Answer Type - Check the type of answer desired;
    • Check before submit - Make the answer mandatory for survey submission;
    • Active - Check this option to enable this question on the survey;
  7. Press ADD;
  8. Save when done (bottom right corner disk button).
