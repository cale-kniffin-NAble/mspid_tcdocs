: How to allow a remote user to create a deferred support request

Last Modified

Fri May 29 09:36 GMT 2020

Description

  • How to allow a remote user to create a deferred support request/ticket?

Environment

Solution

  1. Login to your Admin Area;
  2. Navigate to Devices;
  3. Click the vertical ellipsis button for the intended device (last column on the list) and click Edit;
    • or click the menu Edit after selecting the device;
  4. A configuration window for the device will appear;
  5. Slide the toggle for can create a deferred support request;
  6. Click on Save (lower right corner);
  7. The new settings can take a couple minutes to take place but restarting the agent will be faster.
