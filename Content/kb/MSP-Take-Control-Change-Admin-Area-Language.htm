: Change Admin Area Language

Last Modified

Thu Jul 19 14:54 GMT 2018

Description

  • How to change the language in use in the Admin Area to English/Portuguese/Spanish?

Environment

Solution

  1. Login to your Admin Area;
  2. Click on the User Menu (top right corner);
  3. Select Change Language;
  4. Set the language as intended and press Save.
