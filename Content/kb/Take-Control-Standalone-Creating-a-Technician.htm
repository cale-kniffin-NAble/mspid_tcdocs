: Creating a Technician

Last Modified

Tue May 21 08:35 GMT 2019

Description

  • How to setup a new login.
  • How to create a new Technician's profile.

Environment

  • /

Solution

  1. Login to your Admin Area;
  2. Navigate to Management > Technicians;
  3. Press Add (top right corner);
  4. Fill out the desired Name, Username (must be a valid email address) and Expiration Date;
    • Review the Permissions and Default Settings tabs to ensure technician permissions and settings are as intended as well.
  5. Save when done (bottom right corner of the creation window);
  6. The password will be set by the technician directly through the SSO invitation email that is sent after the profile is created.
