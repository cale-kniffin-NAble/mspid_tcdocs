Take Control: How to show remote cursor while on a remote session

Last Modified

Tue Aug 11 12:40 GMT 2020

Description

  • I need to know when my partner is moving the remote cursor around to avoid catching them by surprise, how can I view their cursor's location?

Environment

  • / 7.00.00 onward;
    • 7.00.00 onward.
  • :
    • 7.00.00 onward.

Solution

  • While performing a Remote Session:
    1. Go to the Remote Desktop tab;
    2. Click on the drop-down menu to the right of Remote Desktop (looks like a downward facing arrow);
    3. Click the menu View, first option in the drop-down menu;
    4. Select the option ShowRemote Cursor.
      • Alternatively, you can use SHIFT + CTRL + T as a keyboard shortcut to toggle this option on/off.
