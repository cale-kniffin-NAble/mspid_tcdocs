: "Kick Tech Error" trying to force logout a technician

Last Modified

Fri Mar 08 13:01 GMT 2019

Description

  • When trying to force logout a technician the following error message appears: "You do not have permission to disconnect this technician."

Environment

Solution

  • This message shows up when the technician to be kicked doesn't have permissions to be disconnected. In order to change this, please proceed as follows:
  1. Login to your Admin Area;
  2. Navigate to Management > Technicians;
  3. Select the technician you wish to disconnect;
  4. Turn on the option Can be disconnected;
  5. Click the Save button.
