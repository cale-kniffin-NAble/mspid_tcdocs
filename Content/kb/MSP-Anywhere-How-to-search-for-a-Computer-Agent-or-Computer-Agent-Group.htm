: How to search for a Computer/Agent or Computer/Agent Group?

Last Modified

Mon Apr 30 17:10 GMT 2018

Description

  • How to search for a specific Computer/Agent or Computer/Agent Group in the Console application?

Environment

Solution

  1. Login to your Console;
  2. Select the My Computers tab;
  3. Type the desired Computer/Agent or Computer/Agent Group name in the Search bar above the list.
  • Partial searching is supported and will filter devices per Computer/Agent name or Computer/Agent Group name in the list;
  • Customer Name, Customer# and EMail field values are also taken into account for the search.
