: Log Locations

Last Modified

Thu Jul 09 13:26 GMT 2020

Description

  • Where can I find the logs?

Environment

  • /
    • Windows
    • macOS
    • Linux

Solution

  • Windows
    • Applet (Remote end):
      • Windows Vista\Windows Server 2008 and above:C:\Users\Bruce\AppData\Local\BeAnywhere Support Express\GetSupport\Logs
      • Windows XP\Windows 2003 Server:C:\Documents and Settings\<username>\Local Settings\Application Data\BeAnywhere Support Express\GetSupport\Logs
    • Agent (Remote end):
      • Windows Vista\Windows Server 2008 and above:C:\ProgramData\GetSupportService\Logs
      • Windows XP\Windows 2003 Server: C:\Documents and Settings\All Users\Application Data\GetSupportService\Logs
    • Agent Installer (Remote end):
      • C:\Windows\Temp\ServerAppletServiceInstaller.install.log
    • Tech Console (Technician's end):
      • Windows Vista\Windows Server 2008 and above: C:\Users\Bruce\AppData\Local\BeAnywhere Support Express\Console\Logs
      • Windows XP\Windows 2003 Server: C:\Documents and Settings\<username>\Local Settings\Application Data\BeAnywhere Support Express\Console\Logs
    • End User Viewer (Client's end):
      • C:\Users\Bruce\AppData\Local\BeAnywhere Support Express\Viewer\Logs
  • macOS
    • Applet (Remote end):

      • /Users/<user>/Library/MSP Anywhere/Applet/Logs

    • Agent (Remote end):
      • /Library/Logs/MSP Anywhere/
      • /Users/<user>/Library/Logs/MSP Anywhere Agent
    • Agent Installer (Remote end):
      • /Library/Logs/MSP Anywhere Installer
    • Tech Console (Technician's end):
      • /Users/<user>/Library/Logs/MSP Anywhere/MSPConsole
      • /Users/<user>/Library/Logs/MSP Anywhere/Viewer MSPA Standalone
    • End User Viewer (Technician's end):
      • Users/<user>/Library/Logs/MSP Anywhere/Viewer End User
  • Linux
    • Agent (Remote end):
      • /opt/tkc_agent_standalone/log
