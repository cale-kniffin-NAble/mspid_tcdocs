: How to perform a remote session from the Admin Area

Last Modified

Wed 13 Jan 12:40 GMT 2021

Description

  • How to perform remote control sessions from the Admin Area web page.

Environment

  • .

Solution

  1. Login to your Admin Area;
  2. Make sure the remote control mode is enabled by clicking the Remote Control Mode Icon near the top right corner of the admin page (the icon should look like an x with 2 curved lines to it's left);
  3. Navigate to Devices;
  4. On the intended device's Action column, click the Connect icon;
  5. The viewer should be launched, allowing you to connect to the remote device.
    • Do note that you still need the Tech Console installed on your device for sessions to launch successfully!
