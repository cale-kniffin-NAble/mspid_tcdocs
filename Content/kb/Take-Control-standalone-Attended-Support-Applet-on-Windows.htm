: Attended Support (Applet) on Windows

Last Modified

Thu Oct 29 18:11 GMT 2020

Description

  • How to generate an Applet executable?
  • How to start attended remote support sessions?

Environment

  • /.

Solution

  • Using the PIN Code and Start Control:
    1. Login to your Take Control Console application;
    2. Press START NEW SESSION;
    3. Copy the generated 9 digit PIN Code;
    4. Direct your client to the Start Control webpage;
    5. They'll need to input your previously generated 9 digit PIN Code;
      • If the first 3 digits are zeros please don't bypass them, they need to be included, otherwise the PIN code will be invalid!
    6. They should now be able to press the DOWNLOAD button;
      • Alternatively, you can provide the customer with the support link instead of having them use the PIN code;
    7. Once downloaded, they'll need to execute the Applet to send a support request;
    8. Accept the incoming Support Request in your Take Control Console Application.
  • Using the direct download link (Windows only):
    1. Login to your Admin Area;
    2. Navigate to Profile > Integration;
    3. Select the 4.Exclusive Link tab;
    4. Provide the Applet Direct Download link to your client;
    5. Once downloaded, they'll need to execute the Applet to send a support request;
    6. Accept the incoming Support Request in your Take Control Console Application.
