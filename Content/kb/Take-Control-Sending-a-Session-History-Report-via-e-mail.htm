Take Control: Sending a Session History Report via e-mail

Last Modified

Thu Mar 14 12:36 GMT 2019

Description

  • How to send a Session History report link via e-mail?

Environment

  • /.

Solution

  1. Login to your Admin Area;
  2. Navigate to Reports > Session History;
  3. Select the intended session from the list;
    • Please left click on the session's line, not the checkbox to the left of each session.
  4. On the bottom right corner, click on Show in Reports Page;
  5. On the reports page, you can e-mail this report by clicking on the Email Report button;
  6. Choose which session details to include on the report and set the intended recipient address under Email;
  7. Click Send
    • An email will be sent from "" with the title "Support session report" to the intended recipient, containing a link to the Session report.
