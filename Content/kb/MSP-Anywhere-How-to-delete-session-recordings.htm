: How to delete session recordings

Last Modified

Thu Jan 18 16:17 GMT 2018

Description

  • How to delete session recordings

Environment

Solution

  1. Navigate to your Admin Area
  2. Go to REPORTS>Session History
  3. Click on the desired session
  4. Select the Video tab
  5. Select the recording(s) at hand.
  6. Hit DELETE
