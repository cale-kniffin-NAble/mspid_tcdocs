: How to Set Gmail as the Default Email client

Last Modified

Fri Mar 22 10:00 GMT 2019

Description

  • You can send an Applet or Agent download link by email to a customer from the Take Control Tech Console. This will invoke the default mail client application registered in Windows.
  • If you don't have an Email Client installed on your computer (Outlook, Firebird,...), and you wish to use Gmail as your default email client, you can do so by following the steps described below.

Environment

  • Tech Console
  • Windows

Solution

  1. Open Chrome and login to Gmail
  2. With Gmail opened, press Ctrl+Shift+J to open up the JavaScript Console.
  3. Paste the following command in the JS console: navigator.registerProtocolHandler("mailto", "https://mail.google.com/mail/?extsrc=mailto&url=%s", "Gmail");
  4. Chrome will ask if you want to set Gmail as default for mailto protocols, click Yes.
  5. Test this by going to the Tech Console, clicking Start New Session then click EMAIL.
  6. It will open a new browser tab with Gmail and the email already prepared.
