Change Permission Request Text in Take Control policy

Last Modified

Wed Aug 05 08:57 GMT 2020

Description

  • In Settings > Take Control > Policy, is it possible to change the Security Permission Request Text?
  • Currently it says, " A technician would like to connect ..."
  • It would be nice to personalize that with my name or company name

Environment

Solution

  • This is an option within Take Control policy
  • Only applies to and not
    1. Click on Settings
    2. Select Take Control
    3. Click on Policy
    4. Click on Security
    5. Click box next to Request user permission:
    6. Click inside Permission Request text: box
    7. Type your message
    8. Click Save at bottom
