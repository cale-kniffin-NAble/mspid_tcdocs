Take Control: Unable to logout of Windows User during an Applet Sesson

Last Modified

Mon Jan 28 14:26 GMT 2019

Description

  • When I logout of a Windows user, I lose my remote session to their device.
  • Can I switch between 2 Windows users during an Applet session?

Environment

  • Take Control

Solution

  • As the Take Control Applet runs on the user's session, it is not possible to switch to another windows user or end the current windows user session and keep the Applet running. When ending the windows user's session, the Applet will close and the remote session will be lost.
  • Tto have a permanent install on a device that runs as a service (not requiring a Windows user to be logged in) consider installing a Take Control Agent. For more information, see [%=General.ProductName_tc-sa%]: How to generate an Agent Installer (Unattended Support).
