: Disabling a Technician

Last Modified

Wed 13 Jan 17:40 GMT 2021

Description

  • How to disable a Technician's Profile, removing it from the visible default Technician's list and revoking access to Take Control for that Technician?
    • The Main Administrator's profile cannot be disabled!

Environment

  • /

Solution

  1. Login to your Admin Area;
  2. Navigate to Management > Technicians;
  3. Select the desired Technician's Profile from the list;
  4. Disable the technician's profile by clicking the Enable button, setting it to OFF;
  5. Save when done (bottom right corner).
