How to set a Take Control policy to View Only/Monitor only?

Last Modified

Wed Oct 24 18:06 GMT 2018

Description

  • How to set a Take Control policy for "view only" take control sessions?
  • How to set a Take Control policy to allow you to connect to a machine to view the screen but not actually be able to provide mouse or keyboard inputs on the remote desktop?

Environment

  • Take Control

Solution

  1. Login to
  2. Go to Settings
  3. Go to Take Control
  4. Select Policy
  5. Edit Policy applied to intended devices (or consider creating a New one)
  6. In General tab, check Monitor Only option
  7. Save policy and apply it to intended devices (a Run Checks cycle may be necessary to apply policy)
