Attended Remote Control Conversion to licenses

Last Modified

Wed Nov 30 21:44 GMT 2016

Description

Late last quarter of 2015, Attended Remote Control was discontinued and all partners that has existing license were converted to .

Environment

  • 10.2
  • All versions

Solution

After converting the Attended Remote Control licenses to :

  • License Team sends out an email informing you of the conversion.
  • On the email, the activation code and the activation link are be provided.
  • After following the activation process, you can now log in to the Admin Area to configure the application.
  • It also includes the download link for the technician console.
