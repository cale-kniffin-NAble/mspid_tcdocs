How to add licenses to Take Control/

Last Modified

Tue Sep 15 14:59 GMT 2020

Description

  • How to add licenses to Take Control/

Environment

Solution

  1. Login to Take Control Admin Area
  2. Click on the username on the top right section of the Admin Area to open the User Menu
  3. Select Licensing Information
  4. Select the Details tab
  5. Select Add licenses
  6. Navigate through the e-commerce process in a new tab to add new licenses.
