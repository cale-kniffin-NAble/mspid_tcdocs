Can't install Take Control Tech Console or Agent in Mac OS X

Last Modified

Tue Nov 20 12:34 GMT 2018

Description

  • When attempting to run the macOS Take Control console install or the Agent install, a version error occurs;
  • Error: This Application Requires Mac OS X 10.9 or newer.

Environment

  • /;
  • macOS X 10.8 or prior version.

Solution

  • Take Control and all of it's features are only compatible with macOS X 10.9 or later versions and will not allow an install on prior versions, this holds true for the Console install, Agent install and Applet execution.
