Take Control Agent Installation Error: "It is not possible to validate the installer at this time"

Last Modified

Mon Jan 28 19:38 GMT 2019

Description

  • Attempting to run the Take Control Agent installer results in a validation error:
    • Error: "It is not possible to validate the installer at this time".

Environment

  • /.

Solution

  • This error occurs when the initial process of the Agent installer validating it's connectivity fails.
  • Usually this is a situational occurrence and pressing Ignore will allow the installation to proceed without any issues.
  • Once the installation completes, should the Agent not read "Service ready" as intended, please attempt the following:
    • Repair your Microsoft XML parser installation: https://www.microsoft.com/en-us/download/details.aspx?id=15697
    • Ensure that communications in the required ports aren't blocked: [%=General.ProductName_tc-sa%]: Ports and Domains/Firewall and AV Exclusions
    • Reach out to our technical support with the Agent logs found under: C:\ProgramData\GetSupportService\Logs
