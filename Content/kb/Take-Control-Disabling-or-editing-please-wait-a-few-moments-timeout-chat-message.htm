Take Control: Disabling or editing "please wait a few moments" timeout chat message

Last Modified

Wed May 01 16:08 GMT 2019

Description

  • How to disable the "Please wait a few moments..." inactivity message that shows up in chat during a session?
  • Can we change the message or it's timer?

Environment

  • .

Solution

  1. Login to your Admin Area;
  2. Navigate to Profile>Applet Setup;
  3. Select the MESSAGES tab;
  4. Set Timeout to 0 seconds and erase the Message;
  5. Save when done using the button in the bottom right corner.
