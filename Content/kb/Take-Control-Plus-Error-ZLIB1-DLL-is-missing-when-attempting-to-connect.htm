: Error ZLIB1.DLL is missing when attempting to connect

Last Modified

Tue Nov 27 11:52 GMT 2018

Description

  • When attempting to connect to a remote machine, a pop up appears indicating the following error:
  • Error: "The program can't start because ZLIB1.dll is missing from the computer. Try to reinstalling the program to fix the problem.".

Environment

Solution

  1. Uninstall the Take Control Agent on the afflicted device
  2. Delete the folders GetSupportService located under:
    • C:\ProgramFiles (x86)\BeAnywhere Support Express\GetSupportService
    • C:\ProgramFiles\BeAnywhere Support Express\GetSupportService
    • C:\ProgramData\GetSupportService
  3. Restart the device
  4. Attempt a new installation of the Take Control Agent
