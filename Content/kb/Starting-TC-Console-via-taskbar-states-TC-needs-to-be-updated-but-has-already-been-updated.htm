Starting Take Control Console via taskbar icon states that Take Control needs to be updated, but it has already been updated

Last Modified

Thu Sep 12 17:52 GMT 2019

Description

  • Starting Take Control via taskbar icon states that Take Control needs to be updated, but opening it from the Start Menu does not display this message
  • Take Control has already been updated to latest version

Environment

    • Taskbar Icon
    • Windows

Solution

  • Take Control Taskbar Icon is reading old data from previous Take Control version
  • Unpin Take Control taskbar icon, then create a new pin for Take Control taskbar icon
    1. Open Start Menu
    2. Find from the Programs list
    3. Right click
    4. Select Unpin from taskbar
    5. Close and reopen Start Menu
    6. Find from the Programs list
    7. Right click
    8. Select Pin to taskbar
