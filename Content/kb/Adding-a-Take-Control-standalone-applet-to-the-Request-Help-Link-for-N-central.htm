Adding a applet to the Request Help Link for

Last Modified

Description

  • Adding Request Help Link for Take Control to

Environment

  • 10.2 or later
  • /
    • Applet

Solution

  • To enable the Request Help link:
  1. Login to as Product Admin;
  2. Navigate to Administration > Branding;
  3. Select the Request Help tab;
  4. Tick the Show Request Help Box:;
  5. On the Request Help Link: insert the Applet Direct Download link;
    • This can be obtained in Take Control's Admin Area;
  6. Save changes.
  • How to obtain the Applet Direct Download link:
  1. Login to Take Control's Admin Area
    • A Take Control/ active account is required!
  2. Navigate to Profile > Integration;
  3. Click the EXCLUSIVE LINK tab;
  4. Copy the Applet Direct Download link;
