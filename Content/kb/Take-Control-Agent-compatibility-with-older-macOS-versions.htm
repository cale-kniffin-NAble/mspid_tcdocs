Take Control Agent compatibility with older macOS versions

Last Modified

Tue Jan 29 12:24 GMT 2019

Description

  • The Agent won't run on an Apple desktop running macOS 10.7 or 10.8.
  • Is the Take Control Agent compatible with older macOS devices?
  • Is there an Agent download for older iMacs?

Environment

  • /.
  • macOS 10.8, 10.7 and older.

Solution

  • Take Control Agent installs for macOS are compatible only with OS X Mavericks 10.9 on-wards.
  • We cannot provide a version for older macOS systems.
