How do I remote into a computer without end user knowing?

Last Modified

Wed May 27 19:39 GMT 2020

Description

  • How do I remote into a computer without end user knowing?

Environment

Solution

  • does not have a policy option for this
  • does have option to enable/disable "Show Visual Indicator"
    • Go to Settings > Take Control > Policy > Security tab then uncheck "Show Visual Indicator".
