Take Control: How to edit email templates and supported tags?

Last Modified

Thu Dec 05 13:47 GMT 2019

Description

  • How to edit email templates?
  • What are the supported tags on email templates?
  • How to make session reports mandatory on every remote session?

Environment

  • /

Solution

  1. Go to your Admin Area;
  2. Navigate to Profile > Email Templates;
  3. Select the following tabs as desired:
  • INSTALL LINK - This template is used when you e-mail your client the Take Control Agent install link. The install link can be used to setup Take Control as a Windows service on the computers of your clients so that you can have unattended access.
    • Supported tags:
      • <#link_install#> Take Control Agent installation link.
      • <#tech_name#> Name of the technician sending the message.
  • SUPPORT LINK - This template is used by technicians when connecting to clients via e-mail. The customer uses the provided link to start a support session.
    • Supported tags:
      • <#link_applet#> Take Control Applet download link.
      • <#tech_name#> Name of the technician sending the message.
      • <#company_name#> Name of the Helpdesk Center.
  • SESSION REPORT - This e-mail template is used to send a report to your client when you close a support session. If set to Optional, it will be up to the technician to decide to send the report or not. If set to Mandatory, the report is always sent. Reports expire 30 days from its generation.
    • Supported tags:
      • <#link_report#> Take Control Session Report link.
      • <#session_wday#> Day of the week in which the session was performed. (Local Machine Time Used)
      • <#session_date#> Date in which the session was performed. (Local Machine Time Used)
      • <#session_hour#> Hour of the beginning of the support session. (Local Machine Time Used)
      • <#session_duration#> Duration of the support session.
      • <#tech_name#> Name of the technician who performed the session. If the session has been transferred, the name of the first tech will be presented.
      • <#company_name#> Name of the Helpdesk Center.
  • Do not forget to save all email templates changes by clicking Save.
