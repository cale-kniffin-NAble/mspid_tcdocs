Take Control: Applet showing a different Language on the client side

Last Modified

Tue May 21 12:08 GMT 2019

Description

  • By default, the language used in the Applet is set up according to the IP address of the client. You can override this setting and force the use of a specific language at all times.

Environment

Solution

  1. Go to Admin Area;
  2. Navigate to PROFILE > Identification;
  3. Select the tab PIN/CALLING CARDS;
  4. Check the option Override Language Detection:;
    • Do not forget to chose the intended language on the drop down Always use the following language on the Applet;
  5. Click the Save button.
