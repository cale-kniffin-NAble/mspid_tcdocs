: How to change Main Admin account to another technician

Last Modified

Wed Jul 01 15:05 GMT 2020

Description

  • How can I transfer main admin admin account to another technician in Take Control Standalone?

Environment

  • /

Solution

  • Only the Main Admin login is capable of performing this procedure:
    1. Login to your Admin Area as the Main Admin;
    2. Click on the User Menu on the top right corner;
    3. Select the Licensing Information option;
    4. Select the ADVANCED tab;
    5. Under Transfer Master Account, select the intended new Main Admin from the Transfer MasterAccount to drop-down;
      • Only tech accounts with Administrator privileges will show on the drop-down;
    6. Press the Transfer button.

Article Number: 119739

Internal notes: For situations like "We can't login to the Main Admin anymore"/"Main Admin has left the company":1) Confirm the request is valid - request a reply from the current main admin account's email with a written request for this change or verify the request through a call to the company's public contact number (DO NOT CALL the person requesting on their personal contact number - that does not validate the request);2) Obtain the new account name and/or intended username (email);3) Escalate case to Application Engineers/Take Control Support team;?Please note and inform the partner that the process of changing main admin details may take some time as changes need to be performed from our infrastructure's end to their account - if asked for an estimate, typically this can be done up to 2 days after receiving confirmation that the request is valid (coming from the account owner/main admin).Please refrain from at ANY POINT requesting the partner's main admin login password - as this may pose a security risk for the partner due to the privileges afforded to a Main Admin login in Take Control!
