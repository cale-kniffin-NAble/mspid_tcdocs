: How to Create an End User Profile

Last Modified

Wed Sep 16 11:39 GMT 2020

Description

  • Steps on How to create an End User profile.
  • Where does the end user log in?

Environment

Solution

  1. Login to the Admin Area;
  2. Navigate to Management > Technicians;
  3. Click on Add (top right corner);
  4. Set your client's intended Name and Username;
  5. Mark the option for Configure as End User;
  6. Save the newly created profile;
  7. The End User will be sent an SSO Invitation e-mail containing a link that will allow them to setup their First and Last Name, as well as their Password;
    • Note that if the End User's e-mail already exists in 's SSO service, they'll instead be asked to login with their existing username and password.
  8. Select the End User profile from the Technicians list and, under the newly appearing COMPUTERS tab, mark which devices will be accessible to this End User;
  9. Save all changes;
  10. Direct your End User to the End User Area;
    • The End User area is only available on Windows and macOS; Android and iOS do not have this feature available.
    • To go to the End User area follow the steps below:
      1. Navigate to the website;
      2. On the top right corner click Login;
      3. Select Take Control;
      4. Click the User Login button.
  11. Have the End User Download and Install the Viewer addon to their workstation;
  12. Following these instructions, your End User should now be able to select a device to remotely connect to from the End User area;
  • Note that End Users will not be able to login on the Console (desktop application), nor the Admin Area, as their use is strictly restricted to Technicians.
  • The end user feature is not currently available on mobile devices.
  • Internet Explorer is not supported for access to the End User area or Admin Area.
