: Allow restricted user accounts to login to this server

Last Modified

Wed 13 Jan 12:57 GMT 2021

Description

  • Can only login to remote session with remote local Admin Credentials on session with Windows login security requirement!
  • How to allow login with restricted user credentials on session that requires Windows credentials to allow the connection?

Environment

  • /
  • Windows

Solution

  1. Access the Remote Device;
  2. Open the Agent Configurator window (found on the hidden icons tray of the taskbar or by searching for Take Control Agent Configuration);
  3. Press the Advanced Setup button, on the top right of the Agent Configurator Window;
    • If the Advanced Setup button is greyed out, press the Enter Administration Mode on the top right corner to enable Admin View.
  4. Click on Security;
  5. Select the authentication method Windows Account;
  6. Check the box Allow restricted user accounts to login to this server;
  7. Click on APPLY.
