: How to Install the Take Control Tech Console in a Flash Drive

Last Modified

Thu Apr 25 15:29 GMT 2019

Description

  • Can the Tech Console be installed in a Flash Drive?

Environment

  • /
  • Windows

Solution

  • How to install the Tech Console in a Flash Drive:
  1. Go to Take Control: Downloads.
  2. Download the Take Control Tech Console;
  3. Run the Installer;
  4. Click I Agree;
  5. Select Both from the Web and Desktop;
  6. Select the second option Install to a pen drive;
  7. From the dropdown menu select the device to where you want to install the Tech Console;
  8. Proceed with the installation.
  • After the installation is complete you may start using the Tech Console from the flash drive.
