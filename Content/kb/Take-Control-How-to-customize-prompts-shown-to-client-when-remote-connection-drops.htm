Take Control: How to customize prompts shown to the client when the remote connection drops

Last Modified

Wed Feb 20 16:50 GMT 2019

Description

  • How to define a message to display to the client when the Internet connection drops during a session.

Environment

Solution

  1. Navigate to your Admin Area;
  2. Go to Profile > Applet Setup;
  3. Under the No connections zone set to On the switch and define a message to be shown;
  4. Click Save.
