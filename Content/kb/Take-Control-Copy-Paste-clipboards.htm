: Copy/Paste clipboards

Last Modified

Tue 12 Jan 09:54 GMT 2021

Description

  • What are the clipboard sharing options available during Take Control sessions?

Environment

  • /.

Solution

  • During a remote support session:
  1. Select he drop-down menu to the right of Remote Desktop (the icon for which looks like a downward facing arrow near the top left corner of the window);
  2. Click on Interactions;
  3. Hover over the Copy/Paste submenu and select either:
    • Get Remote Clipboard: To overwrite your local device's clipboard value with the remote device's clipboard value.
    • Send Local Clipboard: To overwrite the remote device's clipboard value with the local device's clipboard value.
    • Insert Copied Password: To paste a previously copied password.
  4. Under the Interactions menu, hover over the Shared Clipboard submenu and make sure one of the following options is selected accordingly:
    • Disabled: shared clipboard disabled - copied text on the local and remote ends will not be shared between them;
    • Local to Remote: only enables synchronizing the local clipboard value to the remote clipboard on copy;
    • Remote to Local: only enables synchronizing the remote clipboard value to the local clipboard on copy;
    • Bidirectional: Enables synchronizing both the local clipboard value to remote clipboard and the remote clipboard value to the local clipboard on copy;
    • Note that you can always set a new option as the default for the current Console by choosing the Save Current Setting as Default option - by default, Bidirectional will be enabled.
  5. Under the Interactions menu, hover over the Clear Clipboard when the Session Ends submenu and make sure one of the following options is selected accordingly:
    • Disabled: The clipboard won't be cleared, locally or remotely, when the session ends (this is the default value);
    • Local Only: The clipboard on the local end will be cleared when the session ends;
    • Remote Only: The clipboard on the remote end will be cleared when the session ends;
    • Local and Remote: Both clipboards (local and remote) will be cleared when the session ends.
  • The clipboard transfer only supports Western/Latin characters, unsupported copied text will appear listed as "?" when pasted from the remote/local clipboard to the local/remote clipboard, respectively
