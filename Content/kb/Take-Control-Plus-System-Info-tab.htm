Description

  • How can I review hardware details of the remote device during a remote session?
  • How to review Memory and Processor usage of the remote device during a remote session?

Environment

  • .

Solution

  1. Start a remote session to the intended device;
  2. Press the System Info tab;
  3. Press the AUDIT button;
  4. After a few moments, the System details should populate with the audit results.
