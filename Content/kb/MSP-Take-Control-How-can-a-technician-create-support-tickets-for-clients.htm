: How can a technician create support tickets for clients?

Last Modified

Mon Feb 26 16:18 GMT 2018

Description

  • How can a technician create support tickets for clients?

Environment

Solution

  1. Login on the Tech Console
  2. Select the MY COMPUTERS tab
  3. Right-click the intended agent
  4. Select the option Create Support Ticket - causing a browser tab to open with a form?
  5. Fill the form accordingly
  6. Click the CREATE TICKET button
  7. The ticket will be created and will appear on your Tech Support Console's Inbox menu and on your Admin Area under REPORTS>Tickets
