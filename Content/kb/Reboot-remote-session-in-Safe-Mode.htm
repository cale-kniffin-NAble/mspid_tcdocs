Take Control: Reboot remote session in Safe Mode

Last Modified

Thu Nov 26 13:14 GMT 2020

Description

  • This article details the steps on how to reboot the remote computer into Safe Mode with Networking.

Environment

  • Take Control Standalone
  • Take Control

Solution

  1. On the remote session window, go to Computer > Restart and Shutdown, or press Ctrl + Backspace.
    It will open the following options:

  2. The Safe restart and reconnect option will restart the remote machine in Safemode with Networking.


Note: This would require a manual log-on from the remote end.
