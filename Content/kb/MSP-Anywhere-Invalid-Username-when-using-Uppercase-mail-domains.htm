: Invalid Username when using Uppercase mail domains

Last Modified

Wed Oct 24 18:01 GMT 2018

Description

  • Invalid E-mail; unable to create a new technician with an uppercase username

Environment

  • Admin Area

Solution

  • For compatibility purposes (such as compatibility with SSO), you are not able to register technician usernames that have upper case letters in the domain name
  • Input username as "mail@domain.com" instead of "Mail@Domain.com"
