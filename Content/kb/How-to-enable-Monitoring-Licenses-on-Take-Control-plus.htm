How to enable Monitoring Licenses on

Last Modified

Mon Oct 12 09:08 GMT 2020

Description

  • Instructions on how to enable Monitoring Licenses on ;

Environment

Solution

  • Each account has 10 Monitoring Licenses available;
  • To enable a Monitoring License, please proceed as follows:
  1. Login to your Admin Area;
  2. Go to Devices;
  3. Select the target device;
  4. Click the Edit button;
  5. Under the Permissions section, enable the toggle Enable monitoring;
  6. Click Save.
