Take Control security and encryption standards

Last Modified

Mon Jun 01 14:49 GMT 2020

Description

  • What security encryption is used within Take Control connections.

Environment

  • /
    • Take Control

Solution

  • Take Control sessions make use of the Elliptic Curve Diffie-Hellman (ECDH) key agreement scheme to enable secure shared secrets between two endpoints, even before the session is established, to help keep sessions secure;
  • All sessions are secured with the Rijndael Advanced Encryption Standard (AES), using a 256-bit cipher, to help keep data secure both in transit and at rest;
  • We also use FIPS 140-2 certified OpenSSLr modules to help keep sessions safe from prying eyes via advanced cryptographic protocols and packet signing techniques.
