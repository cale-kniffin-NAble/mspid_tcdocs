: How to disable a Department

Last Modified

Tue May 05 14:19 GMT 2020

Description

  • How to disable a technician Department.

Environment

  • /

Solution

  1. Login to the Admin Area;
  2. Navigate to Management > Departments;
  3. Select the intended department from the Departments list;
  4. Slide the toggle Enable this Department to disable mode;
  • It is not possible to disable departments that have technicians assigned to them;
  • All technicians must be unassigned first in order to disable a given department.
