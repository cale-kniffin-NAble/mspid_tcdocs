: Not able to generate PIN Codes/Not Allowed to generate PIN sessions

Last Modified

Thu May 09 10:21 GMT 2019

Description

  • Attempting to generate PIN Codes/Start a New Session from the Console returns an error.
  • Error: You are not allowed to generate PIN sessions!

Environment

  • /.

Solution

  1. Login to the Admin Area;
    • Please note that you will need sufficient permissions to access the admin area and change technician settings.
    • If you are not an admin - please reach out to your account's administrator.
  2. Click on Management > Technicians;
  3. Click on the Intended Technician's profile from the list;
  4. Select the PERMISSIONS tab;
  5. Select the GLOBAL tab;
  6. Make sure that Generate PIN Codes is enabled (should appear as green);
    • Review the remaining settings for the Permissions tab to ensure your Technician has all the intended privileges.
  7. Save when done (bottom right corner).
    • The technician settings will change once they login to the Console after the changes have been performed.
