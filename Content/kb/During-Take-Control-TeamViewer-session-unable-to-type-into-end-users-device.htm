During session unable to type into end users device

Last Modified

Wed May 20 08:05 GMT 2020

Description

  • During session unable to type into end users device

Environment

Solution

  • On device running Viewer
    1. Open Viewer
    2. Select Extras
    3. Select Options
    4. Select Advanced
    5. Select Show advanced options
    6. Scroll down to heading "Advanced settings for connections to other computers"
    7. Select the drop-down for Access Control
    8. Select Full Access
    9. Click OK
  • On host device
    1. Right-click the TeamViewer system tray icon
    2. Select Options
    3. Select Advanced
    4. Select Show advanced options
    5. Scroll down to heading "Advanced settings for connections to this computer"
    6. Select that the drop-down for Access Control
    7. Select Full Access
    8. Click OK
