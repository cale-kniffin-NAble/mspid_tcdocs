: Hide System Info/Request Details automatic pop up

Last Modified

Wed Apr 29 16:53 GMT 2020

Description

  • Instructions on how to hide System Info/Request Details pop up window when launching a remote session.

Environment

  • /

Solution

  • While on a remote session:
  1. Click the Preferences icon (vertical ellipsis) on the top right corner of the window;
  2. Uncheck the option Open System Details when the Session starts;
  • Next remote sessions will manifest the defined behavior;
  • This is a per Viewer setting, meaning this will need to be setup individually on each Viewer.
