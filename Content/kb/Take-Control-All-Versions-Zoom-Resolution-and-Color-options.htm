Take Control (All Versions): Zoom, Resolution and Color options

Last Modified

Wed Feb 26 09:40 GMT 2020

Description

  • Fix for when remote Desktop resolution is too small or large
  • How to Zoom in or out of the Remote Desktop?
  • How to change the Remote Desktop resolution during a session?
  • How to change the default color from Low to True color?
  • How to adapt the session for a low bandwidth connection?

Environment

  • /
    • Take Control

Solution

  • During a remote session:
    1. Select the Remote Desktop tab;
    2. Select the View menu;
    3. Use one of the following options as needed:
      • Size: Set the desired scale (zoom) of the remote desktop display (100% being 1:1);
      • Color: Select the color saturation you intend to experience during a remote session (from Grayscale to True Color);
      • Change Remote Resolution: Alter the remote desktop resolution to a desired size (usually the same as your local resolution);
      • Full Screen: Take the Remote Desktop into Fullscreen (you can enter/leave Full Screen by using the hotkeys Shift+Ctrl+F).
  • Other options that may impact visual quality of the sessions include the performance settings for Optimize Desktop Settings while in Session and Limit Image Capture - which can be set in the Agent Settings or Policy Settings for a device.
