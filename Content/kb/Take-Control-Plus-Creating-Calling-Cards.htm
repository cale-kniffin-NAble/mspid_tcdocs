: Creating Calling Cards

Last Modified

Wed Jan 09 14:00 GMT 2019

Description

  • How to create Calling Cards?

Environment

Solution

  • Using the Admin Area
  1. Login to your Admin Area;
  2. Navigate to Management > Calling Cards;
  3. Select the Calling Card batch (from the list on the left side, or Add a new batch);
  4. Select the Active Cards tab;
  5. Press Add (lower right corner);
  6. Choose the duration, expiration date and code numbers that you want to create and press Add;
  7. When selecting the Calling Card group's Active Cards, those codes should now be listed on the right side - these are the codes your clients will use instead of a PIN Code
  • Using the Tech Console
  1. Login to your Technician's Console application;
  2. Mark NewCallingCard rather than New Session (on the right side of the window);
  3. Press the GENERATECALLINGCARD button;
    • This Calling Card was placed in your default Calling Card group, if you'd like to edit it's settings consult it in the Admin Area, under Management > Calling Cards.
  4. You may now COPY or EMAIL the Calling Card Link or ID to your client for use instead of a PIN Code.
