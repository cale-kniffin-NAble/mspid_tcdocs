: Disconnect Sessions on Timeout

Last Modified

Mon Dec 23 16:07 GMT 2019

Description

  • How to set remote sessions to automatically disconnect after a period of idle time?

Environment

Solution

  • Via Admin Area:
    1. Login to your Admin Area;
    2. Navigate to Profile > Identification;
    3. Select the SECURITY tab;
    4. Under the area Session Timeout check the option Disconnect sessions if idle more than;
    5. Select the intended timeout on the drop-down box;
    6. Save when done.
  • During a Remote Session:
    1. Click the vertical ellipsis icon, located on the top right corner of your viewer;
    2. Hover over Session Inactivity Timeout;
    3. Select the desired timeout.
      • If you intend to save the defined timeout as default:
      1. Click the vertical ellipsis icon > Session Inactivity Timeout > Save Current Setting As Default;
      2. A window will show up to double check the action, click the YES button.
