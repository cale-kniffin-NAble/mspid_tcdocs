: Ticket Status Visibility

Last Modified

Mon Mar 12 15:06 GMT 2018

Description

  • How does each Ticket Status influence Console Inbox visibility per Technician

Environment

Solution

  • Through either the Admin Area (Reports > Tickets) or the Console (Inbox), Tickets can be set to one of the following Status:
    • Opened: Visible to all technicians with Ticket access permissions, will appear in their Console Inbox;
    • In Progress: Visible to the assigned technician or department, will appear in their Console Inbox;
    • Close: Not visible to the assigned technician or department, will not appear in their Console Inbox.
