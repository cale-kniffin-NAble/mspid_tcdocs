Take Control: How to verify currently running sessions

Last Modified

Fri Oct 04 10:55 GMT 2019

Description

  • How to check currently running sessions on ?
  • Where to view currently pending sessions?

Environment

Solution

  • From the Admin Area:
    1. Login to your Admin Area;
    2. Select the Sessions menu.
      • This feature is only available for licenses.
  • From the Console:
    1. Login to the Take Control Console;
    2. Select the PENDING REQUESTS tab.
      • Please note that request visibility will depend on Technician permissions - specifically which queues requests are assigned to and which queues or departments are assigned to the Technician.
