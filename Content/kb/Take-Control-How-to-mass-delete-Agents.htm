Take Control: How to mass delete Agents?

Last Modified

Fri Jun 07 18:45 GMT 2019

Description

  • If I'm looking to remove several Agents across a number of Groups, how can this be done?

Environment

  • /.

Solution

  1. Login to your Admin Area;
  2. Navigate to Devices;
  3. Left-Click on the All Devices filter from the list to the left;
  4. Check the intended devices from the list to the right;
  5. Left-Click on the the now visible More option (appearing on the north pane above the list);
  6. Select Delete;
  7. Please review the deletion prompt carefully and, should you wish to confirm the deletion, press Yes (to the left of No, which is underlined);
  8. The devices will be removed from your list and the Agents will receive a command to silently uninstall themselves from the client devices momentarily.
