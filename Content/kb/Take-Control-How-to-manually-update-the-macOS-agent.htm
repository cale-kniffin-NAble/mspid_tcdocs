Take Control: How to manually update the macOS agent

Last Modified

Tue Nov 20 13:03 GMT 2018

Description

  • How to manually update the macOS remote agent.

Environment

  • /
  • macOS

Solution

  1. Navigate to startcontrol;
  2. Select the MAC tab;
  3. Click the Download Agent button;
  4. Execute the downloaded BAManagement installer;
  5. Select the option Update the current Agent;
  6. Click the UPDATE button;
  7. The Take Control Agent should proceed to update as expected.
