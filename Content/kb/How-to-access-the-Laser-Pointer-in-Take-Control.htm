How to access the Laser Pointer in Take Control

Last Modified

Tue Mar 03 17:46 GMT 2020

Description

  • How to access the Laser Pointer in Take Control
  • Where is the laser pointer in Take Control?

Environment

Solution

  1. Laser Pointer can be accessed two different ways while in Take Control session:
    1. Access Laser Pointer via Hotkey
      1. Press Shift+Ctrl+K to activate Laser Pointer
      2. Press Shift+Ctrl+K again to disable
    2. Access Laser Pointer from Take Control Console Menu
      1. Select Remote Desktop
      2. Navigate to Interactions
      3. Select Laser Pointer
      4. Select Laser Pointer again from Interactions to disable
