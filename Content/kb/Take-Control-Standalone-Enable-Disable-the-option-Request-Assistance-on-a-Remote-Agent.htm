: Enable/Disable the option Request Assistance on a Remote Agent

Last Modified

Tue Jul 28 12:03 GMT 2020

Description

  • How to enable/disable the option Request Assistance on the Agent Configurator Window

Environment

  • /

Solution

We may enable or disable the option Request Assistance on the Agent Configurator of a remote machine, to a single machine or to an entire Group.

  • Enabling or Disabling the Request Assistance button to a Single Machine:
    1. Login on the Admin Area
    2. Go to Devices
    3. Select Select the Device
    4. Click Edit
    5. Enable/Disable the option Can create support requests
    6. Click Save
  • Enabling or Disabling the Request Assistance button to an entire Group:
    1. Login on the Admin Area
    2. Go to Devices
    3. Check the box on the left of Devices, to select the entire group
    4. Click Edit
    5. Enable/Disable the option Can create support requests
    6. Click Save
  • If you want to speed up the propagation of this option to a machine, you may do it through the Admin Area or your Tech Console.
    • Admin Area:
      1. Login on the Admin Area
      2. Select Devices
      3. Select the intended Device
      4. Click More
      5. Select Restart Agent
    • Tech Console:
      1. Login on the Take Control Tech Console
      2. Go to MY COMPUTERS tab
      3. Right-click on the intended Device
      4. Select Restart Remote Service
      • Once the Agent restarts the button Request Assistance should be available and seen on the remote Agent's Configurator window.
