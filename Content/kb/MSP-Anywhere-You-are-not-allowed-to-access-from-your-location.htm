: You are not allowed to access from your location

Last Modified

Mon Feb 26 18:18 GMT 2018

Description

  • You receive the error: "You are not allowed to access from your location." when trying to login at 's Admin Area.

Environment

Solution

  • You have enabled IP restrictions for Admin Area logins and your current public IP address isn't part of the authorized IP list.
  • This can be resolved by:
  1. Login to the Admin Area from an authorized IP address or from the Tech Console.
  2. Go to Profile > Identification > 5. Security
  3. In the I. Access Control section, uncheck Limit the access to Admin Area by IP or adjust the IP addresses in the IP Set field.
  4. Click on the Save All (floppy disk) button
