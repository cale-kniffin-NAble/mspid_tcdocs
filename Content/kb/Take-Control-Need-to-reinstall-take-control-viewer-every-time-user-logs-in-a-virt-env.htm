: Need to reinstall every time the user logs in a virtualized environment

Last Modified

Tue Oct 29 02:09 GMT 2019

Description

  • crashes every time the user logs in.
  • needs to be reinstalled in order to work.

Environment

  • Citrix Environments

Solution

  • In cases where virtualization systems are at place (Windows applications, desktops, and resources virtualization), similar to Citrix or others alike, where user profiles are matched with a virtual image, it is important to ensure the following:
  1. The user profile is properly preserved;
  2. The installation folder isn't tampered with in any way
    • The installation folder is located under C:\Users\[USER]\AppData\Local\Take Control Viewer
    • Please remove and reinstall the viewer if it appears corrupt
