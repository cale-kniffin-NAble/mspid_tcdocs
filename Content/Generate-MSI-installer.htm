Generate an MSI installer

While adding a new device from the Devices page of the Admin Area, select the Microsoft Windows MSI installeroption to mass deploy the Agent to multiple devices.

  1. In the Admin Area, select Devices from the left navigation pane, click Add device and enter a name for the new device.

    2. Navigate to Profile > Agent Setup > General to configure the computer name as the default device name. When this is enabled, the supplied device name is ignored and replaced with the computer name.

  2. In the Operating System section of the new device configuration window, select the Microsoft Windows MSI option.
  3. Enter the Customer Details.
  4. In the Device Expiry section, select Device never expires.
  5. Click Save and the installer downloads in a new tab in your browser.
  6. After the download is complete, open and deploy the MSI file with the appropriate parameters:
    1. Normal silent install: msiexec /i /qn path_to_msi
    2. Silent install of an instance: msiexec /i /qn path_to_msi INSTANCE="instance_name"
    3. Uninstall: msiexec /uninstall path_to_msi

      4. This option requires an MSI on disk.

Make sure to configure a Group Policy, as the group policy definition is what actually deploys and installs the Agent on your devices. The deployment pushes the MSI to a group policy controller which populates to all downstream devices and installs the Agent.
