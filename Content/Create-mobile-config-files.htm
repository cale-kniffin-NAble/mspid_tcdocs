Create Mobile Config files

Add configuration files to use later when connecting to mobile devices. These files provide a way to set up features like Wi-Fi networks or E-Mail accounts.

  1. Select Management > Mobile Config.
  2. Click Add and enter a name and description for the new mobile config file. Click Add to confirm the addition of the new file.
  3. Select the new configuration file from the left side of the page, and make sure the it is enabled.
  4. Select which departments are able to see the config file.
  5. Click Upload File to add the configuration file.
  6. Save your changes.
