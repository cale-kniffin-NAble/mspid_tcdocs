Generate a Tech Performance report

The tech performance report shows key indicators related to the performance of each technician. Filter your results to show stats for the last day, week or month.

Watch this video to learn more about the Technician performance dashboard.

  1. Navigate to Reports > Tech Performance.
  2. Select filters to adjust the data included in the report.
  3. Choose a Chart Measurement option to generate the results at the bottom of the page.
  4. Select either a columns or pie chart to change the style of the graph.
  5. Select Export to export the report for a specific technician or all technicians.
