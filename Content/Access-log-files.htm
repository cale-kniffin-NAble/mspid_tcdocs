Access log files

Log files are crucial to help diagnose issues that occur for your customers. They are also helpful when technicians experience issues and need to contact 's support, provide the Console log files, in addition to the issues pertaining to the remote Applet, or to the remote Agent.

  • For the Applet, customers can right click on the Applet logo in the chat window and select Open log folder.
  • For the Agent, select Options > Show Log folder.
  • For the Agent, use the Send troubleshoot report option in the context menu of the Agent tray icon.
