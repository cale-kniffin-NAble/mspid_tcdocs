Create an end user

End-Users are counted apart from your normal technician licensing, for example you can have a license for 1 Corporate Technician and 10 licenses for end users. Configure your customers as End Users to give them restricted access to their computers and devices with a limited set of features and permissions.

End Users cannot access the Admin Area or any of the Consoles, and are limited to their own End User Area where they have limited access to a number of devices and can perform basic remote control and file transfer tasks.

  1. From the Admin Area, Navigate to Management > Technicians, and click Add.
  2. Enter the Name and Email for the end user, select Configure as End User and click Save.
  3. Click Send invitation to send the SSO (Single Sign On) invitation to the end user.

    4. The new end user will receive an email with instructions on how to set up their SSO account and log in to the End User area.

  4. The Permissions and Default Settings tabs are replaced with the Customers tab, where you can choose which devices the end user can access. Select the Computers tab, and use the Custom Name field to change the name of each computer to which the end user has access.
  5. Save your changes.

 

The usage of the End-User licensing is subject to a Fair Usage Policy which is regularly updated. See the website for more information.
