Configure the proxy on the Agent or Applet

The Agent and Applet are automatically configured, and no manual setup is required in most situations. However, in the event that connectivity problems or incorrect configuration of the proxy machine impairs the automatic setup, some configuration is required:

  1. Run the applet or start the Agent configuration program on a computer with a limited or nonexistent internet connection.
  2. Click Yes when asked if the computer is behind a proxy server.
  3. Click Yes when asked if a dedicated proxy server is present in your network.
  4. Select Use Take Control Proxy Server, and enter the Proxy IP address and port number.
  5. Select Enable Direct Connect support if the proxy is using Direct Connect.

  6. Click Test my proxy settings to verify that the connection to the proxy is successfully established.

  7. Click Apply.
