Call other technicians

Use the Tech Console to call other technicians before or during a support session. This is helpful when there is a need to further discuss a customers issue, or chat before starting a support session.

  1. Navigate to the Chat link in the Technician Console and select the Technicians tab.
  2. Right click the relevant technician and select Start VoIP call with Tech.

A pop-up window opens for the receiver of the call, giving the option to accept or reject the call.

It is also possible to create a conference and call multiple technicians at the same time.
