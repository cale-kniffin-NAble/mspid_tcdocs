Create and manage Calling Cards

A Calling card is a special version of the applet that can be installed on a customer’s computer, and used any time they need remote assistance. Calling Cards have a reference ID that is composed of an alphanumeric prefix, and are used to download the applet.

Configure Calling Cards with an expiration date and a maximum session time. This gives providers the ability to launch special promotions or different types of support for certain products. allows the creation of single Calling Cards, or groups of multiple cards. Manage all of your calling cards from the Admin Area.

What do you want to do?

  • Create a single Calling Card from the console
  • Create and manage Calling Cards from the Admin Area
  • Manage one or more Calling Cards
