Generate an Audit Report

provides the ability to audit one or more devices. Use this feature to obtain reports about the activities performed on the selected devices, as well as detailed information about the devices themselves.

  1. Perform an audit from the Devices page of the Admin Area.
  2. Navigate to Reports > Audits to view the Audit information.
  3. Click Show report document to view the audit report.
