Generate an Audit or Monitoring Report

Use the Audit/Monitoring Report to audit one or more devices and obtain reports about the activities performed on the selected devices, as well as detailed information about the devices themselves.

  1. Navigate to the Devices page of the Admin Area and select a device from the list.

    2. Make sure Monitoring is enabled by editing the device, selecting Enable Monitoring from the Permissions Section of the editor and saving the device.

  2. Click More > Monitoring Report and select the report you would like to generate.
  3. Enter the email address where you would like a copy of the report sent.
  4. Click Select monitoring report to send the report to the specified email address. This also opens the report in a separate tab in your browser.

After running a Monitoring report to audit devices from the Devices page, the report is available to view from the Reports > Audits section of the Admin Area.
